## 功能特性

- **iOS 原生还原**：像素级复刻 iOS 微信 UI，包括气泡比例、间距、导航栏和状态栏。
- **自定义环境**：可修改电量、信号、昵称、头像以及背景图。
- **消息类型**：支持文本、图片和时间戳，支持双向对话模拟。
- **实时预览**：采用 React 状态驱动，所有修改即时反映在预览区域。
- **高清导出**：通过 `html2canvas` 解决 Web 端字体模糊和截断问题，一键生成 PNG。

## 使用说明

1. 在左侧面板修改“对方昵称”和“头像”。
2. 使用添加按钮插入对话。
3. 点击消息列表中的文字可直接修改内容。
4. 满意后点击 **导出高清 PNG** 即可保存图片。

## 技术实现

- **布局**：使用 Tailwind CSS 进行响应式布局，全屏模式适配。
- **截图**：`html2canvas` 配合 `useCORS` 处理跨域头像显示。
- **动画**：使用 `tailwindcss-animate` 实现消息弹出的动效。
