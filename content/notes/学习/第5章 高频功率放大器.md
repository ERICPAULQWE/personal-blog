## 引言：高频功率放大器的应用场景

高频功率放大器（High-Frequency Power Amplifier, HFPA）是无线通信、广播、雷达等系统中不可或缺的关键部件，主要用于将低功率的高频信号放大到足以驱动天线或其他负载所需的功率水平。
<font color="#8064a2">可按标签/系列归档</font>
* **发射机中的应用 (示例：调幅广播发射机)** 
    * 信号源（如话筒经调制器处理后的信号）产生的功率通常很弱。
    * 需要经过一系列放大，最终通过末级**功率放大器 (PA)** 将信号功率提升至目标发射功率。
    * 典型的发射机链路可能包含：主振荡器 -> 缓冲级 -> (倍频级) -> 中间放大级 -> **功率放大级** -> 滤波/匹配网络 -> 天线。
> [!figure]+ Image: 调幅广播发射机方框图
> ![[Screenshot_20250417_201910.jpg]]


* **接收机中的应用 (对比：高频调谐小信号放大器)** 
    * 接收机前端通常使用高频**小信号**放大器 (Low Noise Amplifier, LNA)，其主要目标是放大天线接收到的微弱信号，同时尽可能降低自身引入的噪声，以提高接收灵敏度。
    * 无论是发射机还是接收机中的高频放大器，通常都需要使用**调谐（选频）网络**（如 LC 谐振回路）来选择所需的工作频率，抑制带外干扰和噪声。
    * 超外差接收机利用混频器将不同频率的接收信号变换到固定的中频 (IF)，便于后续统一处理（如滤波和放大），从而提高接收机的选择性和灵敏度。

## 5.1 概述

### 1. 定义与作用
在高频（射频 RF、微波 MW 等）频段，用于显著提升信号**功率**的放大器。

### 2. 主要要求
* **大输出功率 ($P_o$)**: 根据应用需求，输出功率可能从几瓦到数千瓦不等。
* **高效率 ($\eta$)**: 将直流电源提供的功率 $P_{=}$ 尽可能多地转换为有效的高频输出功率 $P_o$，减少能量损耗（主要以热量形式 $P_C$ 散失）。效率是衡量功率放大器性能的关键指标。

### 3. 特点
* **工作频率高，相对频带窄**: 通常围绕某一中心频率工作，带宽与中心频率之比（相对带宽）较小。
* **采用选频网络作为负载**: 输出端通常接 LC 谐振回路或其他形式的带通滤波器，用于：
    * **滤波**: 选择需要的基波频率，抑制谐波失真。
    * **阻抗匹配**: 将负载阻抗变换为适合晶体管工作的最佳阻抗。
* **非线性工作状态**: 为追求高效率，功率管（晶体管或电子管）常工作在非线性状态，如 C 类 (丙类) 或开关状态 (D类、E类等)。这意味着晶体管在一个信号周期内仅导通一部分时间。
* **分析方法**: 由于非线性工作，小信号线性模型不再适用。常采用图解法、基于器件输出/转移特性的折线逼近法或时域波形分析法。

### 4. 工作状态分类 (按导通角 $2\theta$)
导通角是指在一个输入信号周期内，晶体管集电极（或漏极）电流 $i_C$ 不为零的角度范围。

* **A 类 (甲类)**: $2\theta = 360^{\circ}$ ($\theta = 180^{\circ}$)。晶体管始终导通。线性度最好，效率最低 (理论 $\eta_{max} = 50\%$ for transformer load)。
* **B 类 (乙类)**: $2\theta = 180^{\circ}$ ($\theta = 90^{\circ}$)。晶体管导通半个周期。效率较高 (理论 $\eta_{max} = 78.5\%$)，存在交越失真。
* **AB 类 (甲乙类)**: $180^{\circ} < 2\theta < 360^{\circ}$ ($90^{\circ} < \theta < 180^{\circ}$)。介于 A 类和 B 类之间，是效率和线性度的折中。
* **C 类 (丙类)**: $2\theta < 180^{\circ}$ ($\theta < 90^{\circ}$)。晶体管导通时间小于半周期。效率高 (谐振放大器理论 $\eta$ 可接近 100%, 实际约 70%-90%)，但失真严重，输出电流为脉冲状，必须配合选频网络使用。是高频**谐振**功率放大器的常用状态。
* **开关模式 (D类, E类, F类, S类等)**: 晶体管工作在饱和和截止两种状态之间快速切换，理想情况下管耗极小，理论效率可达 100%。

### 5. 高频功放与低频功放的主要差异

| 特性   | 低频功放               | 高频功放                     |
| :--- | :----------------- | :----------------------- |
| 工作频率 | 低 (如音频 20Hz-20kHz) | 高 (MHz, GHz)             |
| 相对带宽 | 宽                  | 通常窄 (调谐放大) 或 宽 (宽带放大)    |
| 负载类型 | 无调谐 (电阻, 扬声器, 变压器) | **选频网络** (LC 谐振回路等)      |
| 工作状态 | A, AB, B (侧重线性度)   | **C**, D, E 等 (侧重**效率**) |
| 输入驱动 | 通常电压驱动，所需功率较小      | 需要一定的**驱动功率**            |
|      |                    |                          |
|      |                    |                          |

### 6. 转移/输出特性与工作点 (以 NPN BJT 为例)
静态工作点 Q 的设置决定了放大器的类别，主要通过基极偏置电压 $V_{BB}$ 控制。$V_{BZ}$ 是晶体管的开启电压 (Base-Emitter turn-on voltage)。

* **A 类**: $V_{BB}$ 使晶体管在无信号时处于放大区中部，$-V_{BB} > V_{BZ}$ (假设 $V_{BB}$ 为负偏压)。
* **B 类**: $V_{BB}$ 使晶体管恰好处于导通边缘，$-V_{BB} \approx V_{BZ}$。
* **C 类**: $V_{BB}$ 使晶体管在无信号时处于截止状态 (发射结反偏)，$-V_{BB} < V_{BZ}$。需要较大的输入信号摆幅才能使其导通。

### 7.高频功率放大器的主要质量指标
#### 1.高频功率放大器的主要特点
(1) 高频功率放大器要求高效率
$P_=$ 直流电源供给的直流功率
$P_o$ 交流输出信号功率
$P_c$ 管子集电极耗散功率
关系：$P_==P_o+P_c$
集电极效率 $\eta_c=\frac{P_o}{P_o+P_c}$
可见：*$P_c↓⇨\eta_c↑$
$P_c=\int_0^{2\pi} v_{\text{CE}} i_C \,d\omega t$ 
#### 降低 $P_c$的措施
* 1.当管子有比较大的$i_C$时，尽量减少这期间的$v_{CE}$
* 2.当$v_{CE}$较大时，尽量减少这期间的$i_{CE}$ ；
* 3.尽量减少$I_c$和$v_{CE}$均不为零的期间

(2) 高频功率放大器一般工作在非线性区
> [!figure]+ [Image: 高频功率放大器工作状态]
> ![[Screenshot_20250417_213504.jpg]]
##### 甲类放大：$-V_{BB}>V_{{BZ}}$ 发射极正偏，集电极反偏
##### 丙类放大：$-V_{BB}<V_{BZ}$ 一般基极直流偏压$V_{BB}$使得发射结反偏
(3) 高频功率放大器要求输入电压要大，同事要求有一定的功率
#### 2.分析方法(折线法)
特性曲线
* 输出特性曲线：基极电流恒定，集电极电流与电压的关系
* 转移特性曲线：集电极电压恒定，集电极电流与基极电压之间的关系
从输出特性曲线看：OP称为临界线
工作状态
* 欠压状态(放大) 交流输出电压低
* 临界状态
* 过压状态(饱和) 交流输出电压高
在临界工作状态，有：$i_C=g_{cr}v_{CE}$
#### 转移特性折线化
从转移特性看，有：$g_c = \left. \frac{\Delta i_C}{\Delta v_B} \right|_{v_{CE}}$
其中$g_C$称为晶体管的跨导
∴$i_C=g_C(v_B-V_{BZ})$  $(v_B>V_{BZ})$  ($v_B为集电极电压V_{BZ}为晶体管截止电压$)

## 5.2 谐振功率放大器的工作原理 

### 1. 基本电路结构

* **有源器件**: 大功率晶体管 (BJT, MOSFET等)，要求高耐压、大电流、高频率特性。
* **输入激励电路**: 提供幅度足够的输入信号 $u_b = U_{bm}\cos\omega t$，并可能包含匹配网络。
* **偏置电路**: 提供直流偏置电压，如基极偏置 $-V_{BB}$ 和集电极电源 $V_{CC}$。对于 C 类，$-V_{BB}$ 通常使发射结反偏。
	* **输出谐振回路 (负载)**: 常用 LC 并联谐振回路。
	    * **选频**: 对基波频率 $\omega$ 谐振，呈现高阻抗 $R_p$；对谐波频率 $n\omega$ 失谐，呈现低阻抗，从而滤除谐波。
	    * **阻抗匹配**: 将实际负载阻抗 $R_L$ 变换为晶体管所需的最佳集电极负载 $R_p$。
	    * **直流馈电**: 电感 L 为集电极提供直流通路。

> ![[Screenshot_20250417_212937.jpg]]

### 2. 工作原理分析 (C 类, 折线模型

#### (1) 集电极电流 $i_c$
* 设输出信号电压$u_b=U_{bm}cos\omega t$ 负载回路谐振于基波$\omega$
* **基极-发射极电压**: $u_{BE} = u_b - V_{BB} = -V_{BB} + U_{bm}\cos\omega t$
* **导通条件**: 
	* 当 $u_{BE} < V_{BZ}$ 时，$i_C=0$
	* 当 $u_{BE} > V_{BZ}$ 时，$i_C=g_c(-V_{BB}+U_{bm}\cos\omega t-V_{BZ})$
* **导通半角 $\theta_c$**: 定义 $i_c = 0$ 时的相位角 (绝对值)，由 $u_{BE}(\theta_c) = V_{BZ}$ 决定：
    $$\cos\theta_c = \frac{V_{BB} + V_{BZ}}{U_{bm}}$$
    $$\theta_c=\cos^{-1}\left( \frac{V_{BB}+V_{BZ}}{U_{bm}} \right)$$
    ($V_{BB}$ 为偏置大小，若为负偏压则 $V_{BB}>0$)。晶体管在 $[-\theta_c, \theta_c]$ 区间导通。C 类工作时 $\theta_c < 90^{\circ}$。
> [!figure]+ [Image: $\theta_c、i_c和u_b之间的关系$]
> ![[Screenshot_20250418_002332.jpg]]
* **电流波形 (折线模型)**: 假设导通时 $i_c = g_c(u_{BE} - V_{BZ})$，其中 $g_c$ 是跨导。
    $$
    i_c(\omega t) = g_c U_{bm} (\cos\omega t - \cos\theta_c), \quad |\omega t| \le \theta_c
    $$
    $i_c$ 是周期性的**尖顶余弦脉冲** (cosine-squared pulse)。峰值$I_{cmax} = i_c(0) = g_c U_{bm} (1 - \cos\theta_c)$。
    标准形式：
    $$
    i_c(\omega t) = \begin{cases} I_{cmax} \frac{\cos\omega t - \cos\theta_c}{1 - \cos\theta_c} & \text{if } |\omega t| \le \theta_c \\ 0 & \text{if } |\omega t| > \theta_c \end{cases}
    $$
* **傅里叶分析**: 将脉冲电流 $i_c$ 分解为直流和各次谐波：
	    $$i_c = I_{c0} + I_{cm1}\cos\omega t + I_{cm2}\cos 2\omega t + \cdots +I_{cmn}\cos n\omega t + \dots$$
    * 直流分量: $I_{c0} =\frac{1}{2\pi} \int_{-\pi}^{\pi} i_c d(\omega t) = \frac{I_{cmax}}{\pi} \cdot \frac{\sin \theta_c - \theta_c \cos \theta_c}{1 - \cos \theta_c}=I_{cmax}\alpha_0(\theta_c)$
    * 基波幅值: $I_{cm1} = \frac{1}{2\pi} \int_{-\theta_c}^{\theta_c} i_c \cos(\omega t) d(\omega t) = I_{cmax} \left( \frac{1}{\pi} \cdot \frac{\theta_c - \sin \theta_c \cos \theta_c}{1 - \cos \theta_c} \right) = I_{cmax} \alpha_1(\theta_c)$
    * n次谐波幅值: $I_{cmn} = \frac{1}{2\pi} \int_{-\theta_c}^{\theta_c} i_c \cos(n\omega t) d(\omega t) = i_{cmax} \left[ \frac{2}{\pi} \cdot \frac{\sin(n\theta_c)\cos \theta_c - \theta_c \cos n \theta_c \sin \theta_c}{n(n^2 - 1)(1 - \cos \theta_c)} \right] =I_{cmax} \alpha_n(\theta_c)$
    $\alpha_n(\theta_c)$ 是仅与 $\theta_c$ 有关的分解系数，可查表得到。
    $I_{c0}, I_{cm1}, I_{cm_{2}},\dots, I_{cmn}$为直流基波和各次谐波的振幅。

#### (2) 集电极电压 $u_{CE}$ 

* **谐振回路作用**: LC 回路对基波 $\omega$ 谐振，阻抗 $Z_p(\omega) \approx R_p$ (高阻纯电阻)；对 $n$ 次谐波 $n\omega$ 失谐，阻抗 $Z_p(n\omega)$ 很低。
	 $\left| \frac{(Z_p)_{n\omega}}{(Z_p)_{\omega}} \right| = \frac{n}{(n^2 - 1)} Q$
* **输出电压 $u_{c1}$**: 仅基波电流 $i_{c1} = I_{cm1}\cos\omega t$ 在 $R_p$ 上产生显著电压
    $$
    u_{c1} = i_{c1} R_p = (I_{cm1}\cos\omega t) R_p = U_{cm}\cos\omega t
    $$
    其中基波电压幅值 $U_{cm} = I_{cm1} R_p$。谐波电压被旁路。
* **总电压 $u_{CE}$**: 集电极对地（发射极）的瞬时电压：
    $$
    u_{CE}(t) = V_{CC} - u_{c1}(t) = V_{CC} - U_{cm}\cos\omega t
    $$
    波形为直流 $V_{CC}$ 上叠加一个反相的余弦波。最低电压 $u_{CEmin} = V_{CC} - U_{cm}$ 出现在 $\omega t = 0$ (电流峰值时刻)。

#### (3) 能量关系与效率

* **直流电源功耗**: $P_{=} = V_{CC} I_{co}$
* **高频输出功率**: $P_{o} = \frac{1}{2} U_{cm} I_{cm1} = \frac{U_{cm}^2}{2 R_p} = \frac{1}{2} I_{cm1}^2 R_p$
* 回路阻抗：$R_P = \frac{V_{cm}}{I_{cm1}} = \frac{V_{CC} - v_{CEmin}}{I_{cm1}} = \frac{V_{cm}^2}{2P_o} (代入P_o=\frac {1}{2}V_{cm}I_{cm1})$
* **集电极耗散功率**: $P_{C} = P_{=} - P_o$。这部分功率转化为热量。
* **集电极效率**:
    $$
    \eta_c = \frac{P_o}{P_{=}} = \frac{\frac{1}{2} U_{cm} I_{cm1}}{V_{CC} I_{c0}}
    $$
    引入 **电压利用系数** $\xi = \frac{U_{cm}}{V_{CC}}$ ($\xi \le 1$) 和 **电流形状系数** $g_1(\theta_c) = \frac{I_{cm1}}{I_{c0}} = \frac{\alpha_1(\theta_c)}{\alpha_0(\theta_c)}$。
    $$
    \eta_c = \frac{1}{2} \xi g_1(\theta_c)
    $$
* **提高效率 $\eta_c$**:
    1.  **增大 $\xi$**: 使 $U_{cm}$ 尽量接近 $V_{CC}$，即 $u_{CEmin} = V_{CC} - U_{cm}$ 尽量小，接近晶体管饱和压降 $V_{CEsat}$。理想 $\xi=1$。
    2.  **减小 $\theta_c$**: 减小导通角，脉冲变窄，$g_1(\theta_c)$ 增大 (见 Page 21 图)，效率提高。但 $\theta_c$ 过小会导致 $I_{c0}, I_{cm1}$ 减小，可能降低输出功率，且需要更高驱动电压。
	**(1)** $\xi = \frac{U_{Cm}}{V_{CC}} = \frac{I_{Cm1}R_p}{V_{CC}}$, $\therefore R_p \uparrow \Rightarrow \xi \uparrow \Rightarrow \eta_c \uparrow$
	**(2)** $g_1(\theta_c) = \frac{I_{Cm1}}{I_{CO}}$, 一般有: $\theta_c \downarrow \Rightarrow g_1(\theta_c) \uparrow \Rightarrow \eta_c \uparrow$
	**(3)** $\eta_c = \frac{P_o}{P_o + P_c} \Rightarrow P_o = \frac{\eta_c}{1 - \eta_c} P_c$

## 5.3 晶体管谐振功率放大器动态分析

晶体管的静态特性：$i_c = f(u_{BE})|_{u_{CE}=\text{常量}}$
晶体管的动态特性：$i_{BE}→i_C→u_{CE}(R_L)$
### 1. 动态特性方程 (动态负载线)
> [!figure]+ 高频功放电路图
> ![[Screenshot_20250418_004754 1.jpg|400]]

描述信号周期内 $i_c$ 与 $u_{CE}$ 的关系。
设$u_b=U_{bm}\cos\omega t$
联立 $u_{BE} = -V_{BB} + U_{bm}\cos\omega t$, $u_{CE} = V_{CC} - U_{cm}\cos\omega t$ 和 $i_c = g_c(u_{BE} - V_{BZ})$ (导通时)，消去 $\cos\omega t$ 和 $u_{BE}$ 得：
$$i_c = g_d (u_{CE} - U_o)$$

> [!info]+  详细推导过程
> $$\begin{aligned}\\联立\begin{cases}u_{BE}=-V_{BB}+U_{bm}\cos \omega t,\\u_{ce}=V_{CC}-U_{cm}\cos \omega t\end{cases}\\由上两式消除\cos \omega t可得：u_{BE} &= -V_{BB} + U_{bm} \frac{V_{CC} - u_{ce}}{U_{cm}} \\\text{又利用晶体管的内部特性关系(折线方程 5.3.3):} \\i_c &= g_c(u_{BE} - V_{BZ}) \\\quad 代入u_{BE}\text{可得:} \\i_c &= g_c\left(-V_{BB} + U_{bm} \frac{V_{CC} - u_{ce}}{U_{cm}} - V_{BZ}\right) \\&= -g_c \left(\frac{U_{bm}}{U_{cm}}\right) \left[u_{ce} - V_{CC} + U_{cm}\left(\frac{V_{BB} + V_{BZ}}{U_{bm}}\right)\right]\\&= g_d (u_{ce} - U_o)\end{aligned}$$


这是一条直线方程，称为**动态负载线**。
* **动态特征线斜率**: $g_d = -g_c \frac{U_{bm}}{U_{cm}}$ (负值)
* **电压轴截距**: $U_o = V_{CC} - U_{cm} \cos\theta_c$  其中$\cos \theta_c=\frac {V_{BB}+V_{BZ}}{U_{bm}}$

### 2. 动态负载线的绘制

在晶体管输出特性曲线上绘制。
* **方法一**: 找截距 $U_0$ 和斜率 $g_d$ 作图。
* **方法二**: 找两点：
* Q点 (静态工作点): 令$\omega t=90^\circ$ 由外部方程可得：
$$Q:\begin{cases}u_{ce} = V_{CC}, u_{BE} = -U_{BB} \\i_c = I_Q = g_c(-U_{BB} - U_{BZ})\end{cases}$$


* A点 (峰值点): 令$\omega t=0^\circ$ $$A:\begin{cases}u_{ce} = u_{C\min} = V_{CC}-U_{Cm}\\u_{be} = u_{b\max} = -U_{BB} +U_{bm}\end{cases}$$
连接 A, Q 两点（实际工作轨迹只在导通时沿此线移动）。

> [!figure]+ Image: 动态负载线画法示意图 (Page 27, Image 8)
> ![[Screenshot_20250418_213548.jpg|300]]
    
### 3. 三种工作状态 (由负载 $R_p$ 决定)
动态特征线：$$i_c=g_d(u_{ce}-U_o)\text{  其中}\begin{cases}g_d = -g_c \frac{U_{bm}}{U_{cm1}} \\U_o = V_{CC} - U_{cm1}\cos \theta_c \\U_{cm1} = I_{cm1} R_P\end{cases}$$
给定 $V_{CC}, V_{BB}, U_{bm}$，负载 $R_p$ 的大小决定了 $U_{cm}$ 和 $g_d$，从而决定工作状态：
* **欠压 (Undervoltage)**
    * $R_p$ 小 $\implies U_{cm}$ 小 $\implies |g_d|$ 大 (线陡)。
    * 峰值点 A 落在放大区，$u_{CEmin} > V_{CEsat}$。
    * $i_c$ 为尖顶余弦脉冲。集电极电压利用不充分，效率、功率利用率低。

* **过压 (Overvoltage)**
    * $R_p$ 大 $\implies U_{cm}$ 大 $\implies |g_d|$ 小 (线缓)。
    * 峰值点 A 进入饱和区，$u_{CEmin} < V_{CEsat}$。
    * $i_c$ 脉冲顶部因饱和而变平或下凹。效率高，但 $i_c$ 失真增大，功率可能下降。

* **临界 (Critical)**
    * $R_p$ 合适。
    * 峰值点 A 恰好落在放大区与饱和区边界上，$u_{CEmin} \approx V_{CEsat}$。
    * $i_c$ 仍近似尖顶余弦脉冲。
    * 通常能获得**最大输出功率 $P_{ocr}$** 和**高效率**，是理想设计目标。
    * 临界电压利用系数: $\xi_{cr} = (V_{CC} - V_{CEsat}) / V_{CC}$。
> [!figure]+ Image: 三种状态下的 $i_c$ 波形和动态负载线 (Page 29-31, Image 9)
> ![[Screenshot_20250418_215041.jpg]]

### 4. 负载特性 (改变 $R_p$)

固定$V_{CC}, V_{BB}, U_{bm}$不变，改变 $R_p$ 时， $P_{D},P_o,P_{c},\eta_c, I_{C0}, I_{C_{0}}, U_{C1}$ 的变化规律：
* **欠压区**:$R_P \uparrow \implies i_{cmax}几乎不变(略减小)\implies I_{C0}, I_{C1} 近似不变 (恒流源特性)\implies U_{c1}=I_{c1}R_P\uparrow\implies P_{=}=V_{CC}I_{C0}几乎不变$$R_{P}\uparrow \implies P_o=\frac {1}{2}U_{C1}I_{C1}\uparrow \implies \eta_c=\frac{I_{c1}U_{c1}}{2I_{co}V_{CC}}\uparrow \implies P_C=P_{D}-P_{o}\downarrow$。
* **过压区**:$R_p \uparrow 进入过压区(余弦脉冲顶部下凹) \implies i_{cmax}\downarrow \implies I_{c0}, I_{c1} \downarrow \implies U_{c1}=I_{c1}R_{P}几乎不变(略有上升)$$P_{=}=V_{CC}I_{C0}\uparrow \implies P_{o}=\frac{1}{2}U_{C1}I_{C1} \downarrow$   $\eta_{c}=\frac{I_{C1}U_{C1}}{2I_{C0}V_{CC}}$和$P_{C}=P_{D}-P_{o}$几乎不变
* **临界点**: $P_o$ 达到峰值。

**注意** ：
* 临界状态输出功率最大$P_o=P_{ocr}$，效率也比较高，可以说是最佳工作状态，常选择此状态作为末极功放输出状态。过压状态虽然效率高但是输出功率小。
* 在欠压状态$I_{C0},I_{CB1}$几乎不变，功放相当于一个恒流源，而过压状态$U_{C1}$几乎不变，相当于一个恒压源。
> [!figure]+ Image: 负载特性曲线图 ($P_o, \eta_c, I_{c0}, I_{cm1}, U_{cm}$ vs $R_p$) (Page 33-34, Image 10)
> ![[Screenshot_20250419_151827.jpg]]

### 5. 调制特性 (改变 $V_{CC}$ 或 $V_{BB}$)

#### (1) 集电极调制 (改变 $V_{CC}$)
固定 $R_p, U_{bm}, V_{BB}$。
* 若从临界开始，$V_{CC} \uparrow \implies$ 进入欠压，$P_o$ 不变。$V_{CC} \downarrow \implies$ 进入过压，$P_o \downarrow$。
* 显然：在欠压区$I_{C0},I_{C1}$几乎不变$P_o$不变
* **结论**: 只有工作在过压区才能有效调制，集电极调幅 (AM) 应工作在**过压区**。

> [!figure]+ Image: 集电极调制特性曲线图 ($I_{cm1}, I_{c0}, P_o, P_C$ vs $V_{CC}$) (Page 37-38, Image 11, Image 12)
 > ![[Screenshot_20250419_152919.jpg]]

#### (2) 基极调制 (改变 $V_{BB}$)
固定 $R_p, U_{bm}, V_{CC}$。
* $-V_{BB} \downarrow$ (偏置由负向正增大) $\implies 导通角\theta_c \uparrow \implies I_{c0}, I_{cm1} 的幅度和宽度\uparrow$, 状态从欠压 $\rightarrow$ 临界 $\rightarrow$ 过压。过压区 $I_{cm1}, U_{cm}$ 增长变缓。
* **结论**: 基极调幅 (AM) 应工作在 **欠压区**。
* 固定$V_{BB}$、增大$U_{bm}$和固定$U_{bm}$、增大$V_{BB}$的情况类似，它们都使基极输入电压$u_{BEmax}$随之增大，对应的集电极脉冲电流$i_c$的幅度和宽度均增大，放大器的工作状态由欠压进入过压 。
* 当谐振功率放大器作为**线性功率放大器** ，为了使输出信号振幅$U_{cm}$反映输入信号振幅$U_{bm}$的变化，放大器必须在U变化范围内工作在欠压状态。
* 当谐振功率放大器用作**振幅限幅器** 时，放大器必须在$U_{bm}$变化的范围内工作在过压状态。

### 6. 放大特性 (改变 $U_{bm}$)

固定 $R_p, V_{CC}, V_{BB}$。
* $U_{bm} \uparrow \implies$ 驱动增强，$\theta_c \uparrow$, $I_{cmax}, I_{c0}, I_{cm1}, U_{cm} \uparrow$, 状态从欠压 $\rightarrow$ 临界 $\rightarrow$ 过压。
* **线性放大**: 需要 $U_{cm}$ (或 $P_o$) 与 $U_{bm}$ 成比例，应工作在 **欠压区** (整个信号范围)。
* **振幅限幅**: 需要 $U_{cm}$ 对 $U_{bm}$ 变化不敏感，应工作在 **过压区** (饱和区)。

## 5.4 高频功放的高频效应

以上对高频功放均是以**静态特性**为基础的分析，只能近似说明和估计高频功放的工作原理，无法反映高频工作时的其它现象。
实际的高频功放电路，晶体管工作在“**中频区**”甚至“**高频区**”，通常会出现（与低频比较）：
* 输出功率 $\downarrow$
* 效率 $\eta_c \downarrow$
* 功率增益 $G_p \downarrow$
* 输入输出阻抗变为复阻抗
### 1. 少数载流子渡越时间效应

* **定义**: 载流子穿过基区所需时间 $\tau$。
* **影响 (当 $f \approx 1/\tau$)**:
    * 基区电荷存储效应， $i_c$ 响应滞后于 $u_{BE}$。
    * $i_c$ 脉冲峰值降低、宽度增加、相位滞后。
    * $I_{cm1}$ 比例下降，导致 $P_o, \eta_c$ 下降。
    * 限制了晶体管的最高工作频率 (与 $f_T$ 相关)。
> [!figure]+ 高频效应图(Page 42)
> ![[Screenshot_20250419_163707.jpg|250]]
### 2. 非线性电抗效应 (结电容)

由于集电结电容是随集电结电压 $u_{ce}$ 变化的非线性势垒电容，其数值可达 (几十到几百 PF)，在高频时构成输出与输入之间的反馈支路，且频率越高反馈越大，会引起不稳定性。
* **$C_{jc}$ (或 $C_{b'c}$)**: 集电结电容，随电压变化，在高频下提供从输出到输入的反馈路径。
    * 可能导致不稳定（振荡）。
    * 通过 Miller 效应影响输入阻抗。
* **$C_{je}$**: 发射结电容（势垒电容+扩散电容），影响输入阻抗和频率响应。
### 3. 发射极引线电感 ($L_e$) 的影响

* 寄生电感 $L_e$ 在高频下产生阻抗 $\omega L_e$。
* 形成串联负反馈，降低增益。
* 使输入阻抗增加感性分量。
### 4. 饱和压降 ($V_{CEsat}$) 随频率增加的影响

当晶体管工作于高频时，其饱和压降随频率提高而增加，即如果在保持同一工作电流时：
$f \uparrow \rightarrow U_{ces} \uparrow \rightarrow U_{c1} \downarrow \rightarrow \xi = \frac{U_{c1}}{V_{cc}} \downarrow \rightarrow \eta_c = \frac{1}{2}\xi g_1(\theta) \downarrow$

> 在相同 $i_c$ 下，更高频率 $f_2$ 对应的饱和压降 $U_{ces}$ (图示中 $u_{ce}$ 轴截距) 更大。

## 5.5 馈电电路和耦合网络

高频谐振功率放大器要正常工作，除了核心的放大器件外，还需要包含：
* **直流馈电线路**: 为晶体管提供合适的静态工作点（偏置）。
* **交流匹配网络**: 将交流信号功率有效地从放大器传输到负载，或在级间传输。
### 1. 直流馈电线路

用于给晶体管的集电极和基极提供直流电压和电流。

**a) 分类**:

* **串联馈电**: 直流电源、匹配网络（或其一部分）和晶体管串联。
* **并联馈电**: 直流电源通过扼流圈等元件与匹配网络、晶体管并联。
两种馈电方式虽然结构不同，但是总有:$$u_{ce}=V_{CC}-U_{cm1}\cos \omega t$$
**b) 集电极馈电电路**: 

* **设计原则**:
    * 直流分量 $I_{CO}$ 的通路：除晶体管内阻外，对外部电路应近似短路，确保 $V_{CC}$ 主要加在管子上，减少外部功耗。 $P_D = I_{CO}V_{CC}$ 是电源提供的总直流功率。  
    * 基波分量 $I_{C1}$ 的通路：只应在负载（或谐振回路）上产生压降，对其他通路应近似短路。
    * 谐波分量 $I_{Cn}$ 的通路：不应消耗功率，对外部电路应尽可能短路。
    
>[!figure]+ 集电极电流各分量示意 Page 3
> ![[Screenshot_20250419_170919.jpg|150]]
* **常用电路**: 
    * 包含高频扼流圈 $L_C$（通直流、阻交流）和高频旁路电容 $C_C$（通交流、隔直流）。
    * 隔直电容 $C_{C1}$ 用于连接负载，允许交流通过，阻止直流。
    * 无论是串联还是并联馈电，在理想情况下，集电极电压波形仍满足 $u_{ce} = V_{CC} - U_{cm1} \cos \omega t$

>$C_{C1}$隔直电容：通交流隔直流
>![[Screenshot_20250419_171059.jpg|400]]

**c) 基极馈电电路**:

* 同样有串联和并联形式。

* **自给偏压**: 由于单独设置基极电源 $V_{BB}$ 不方便，常用自给偏压方式。
    * **方法1 (并馈)**: 利用基极直流电流 $I_{BO}$ 在基极电阻 $R_B$ 上的压降产生偏置 $V_{BB} = I_{BO} R_B$
    * **方法2 (并馈)**: 利用 $I_{BO}$ 在基区体电阻 $r_{bb'}$ 上的压降产生偏置 $V_{BB} = I_{BO} r_{bb'}$。此偏压通常很小且不稳定。
    * **方法3 (串馈)**: 利用发射极直流电流 $I_{EO}$ 在发射极电阻 $R_E$ 上的压降产生偏置 $V_{BB} = I_{EO} R_E$。这种方式通过负反馈能自动稳定工作点。

### 2. 高频功放的耦合回路（匹配网络）

高频功放都要采用一定的耦合回路，以使输出功率能有效地传
输到负载（下级输入回路或天线回路），一般说来，放大器与负载
之间的耦合可采用下图所示的四端网络来表示。

>[!figure]+ 通用四端匹配网络示意图
>**输入匹配网络或级间耦合网络** : 是用以与放大器的输入端相连接
>**输出匹配网络** : 是用以输出功率至天线或其他负载
>![[Screenshot_20250419_172745.jpg|400]]

**a) 作用**:

* **阻抗匹配**: 将负载阻抗 $R_L$ 变换为放大器所需的最佳负载阻抗 $R_P$，以实现最大功率传输。
* **滤波**: 抑制工作频率（基波）以外的谐波频率和其他不需要的频率成分。
* **功率合成/隔离**: 在多个器件向同一负载输出功率时，保证有效传输并减少器件间的相互影响。

**b) 匹配的概念**:

* **线性电路**: 最大功率传输条件为负载阻抗等于信号源内阻的共轭，即 $Z_L = Z_S^*$；若均为纯阻，则 $R_L = R_S$。

>[!figure]+ 线性电路共轭匹配 Page 8
>![[Screenshot_20250419_173116.jpg|150]]
* **非线性电路 (如丙类功放)**: “匹配”是指将实际负载 $R_L$ 通过匹配网络变换后，折合到晶体管输出端呈现的等效负载电阻 $R_L'$ 应等于放大器为输出所需功率 $P_o$ 而要求的**最佳负载电阻** $R_P$。
    * $R_P = \frac{(V_{CC} - V_{CES})^2}{2 P_o}$ （近似公式，其中 $V_{CES}$ 是晶体管饱和压降）

**c) 输出匹配电路示例 (并联谐振回路型)**:

介于放大器与天线回路之间的$L_1C_1$回路就叫做中介并联谐振回路。$R_A、C_A$分别代表天线的辐射电阻与等效电容；
$L_2、C_2$为天线回路的调谐元件。它们的作用是使天线回路处于串联谐振状态，以使天线回路的电流$I_A$达到最大值，亦即使天线辐射功率达到最大。
从集电极向右方看去可以等效为一个并联谐振回路,其中$R_p$为折合到晶体管输出回路的等效负载 。

>[!figure]+ 并联谐振回路型匹配电路及折合等效谐振回路
>![[Screenshot_20250419_174406.jpg|450]]
* 常使用一个中间并联谐振回路（$L_1C_1$）耦合到负载（如天线回路 $R_A, C_A, L_2, C_2$）。
* 天线回路自身调谐在串联谐振状态，使天线电流 $I_A$ 最大。
* 天线回路会等效一个电阻 $$r' = \frac{\omega^2 M^2}{R_A}$$ 到 $L_1C_1$ 回路上。
* $L_1C_1$ 回路（考虑自身损耗电阻 $r_1$ 和等效负载 $r'$）的总谐振阻抗为 $$R_P' = Q_L \omega L_1 = \frac{L_1}{C_1(r_1 + r')}= \frac{L_1}{C_1(r_1 + \frac{\omega^2 M^2}{R_A})}$$，其中 $Q_L = \frac{\omega L_1}{r_1 + r'}$ 是有载品质因数。
* 若晶体管通过抽头接入 $L_1C_1$ 回路（接入系数为 p），则晶体管看到的最终等效负载为 $$R_p = p^2 R_P' = p^2 \frac{L_1}{C_1(r_1 + \frac{\omega^2 M^2}{R_A})}$$
* 通过调整互感 $M$ 和接入系数 $p$ 可以实现阻抗匹配 ($R_p$ 达到最佳值)。

>[!figure]+ 带抽头的并联谐振回路 Page 11
>![[Screenshot_20250419_175253.jpg|300]]

**d) 回路传输效率**:
由于高频功率放大器工作在非线性(丙类)工作时，放大器的内阻变动剧烈：导通时，内阻很小；截止时内阻近于无穷大。因此输出电阻不是常数。所谓线性电路的阻抗匹配(负载阻抗与电源内阻相等)概念也就失去了意义。
* 中介回路的传输效率 $\eta_k = \frac{P_{out}}{P_{in}} = \frac{I_k^2 r'}{I_k^2 (r_1 + r')} = \frac{r'}{r_1 + r'}$。
* 传输效率也可以用空载品质因数 $Q_O$ 和有载品质因数 $Q_L$ 表示： $$\eta_k = \frac{r'}{r_1+r'}=1 - \frac{r_1}{r_1 + r'} = 1 - \frac{Q_L}{Q_o} =1-\frac{R_{P}'}{R_{P_{o}}'}$$ $Q_o = \frac{\omega L_1}{r_1}=\frac{R_{P_{0}}}{\omega L_1}$  $Q_{L}=\frac{\omega L_1}{r_1+r_1'}=\frac {R_{p}'}{\omega L_1}$ 
	$R_{Po}'=空载回路谐振阻抗=\frac {L_1}{C_{1}r_1}$  $R_{P}'=有载回路谐振阻抗=\frac{L_{1}}{C_{1}(r_1+r_1')}$

* 高效率要求 $Q_o$ 尽可能大（回路自身损耗小），$Q_L$ 尽可能小。
* 但为了良好的滤波特性（抑制谐波），又要求 $Q_L$ 足够大。
* 实际设计中需要折衷，通常 $Q_L$ 取值不小于 10。

## 5.6 丁类(D类)高频功率放大器

### 1. 背景

* 甲、乙、丙类放大器通过减小导通角 $\theta_c$ 来提高效率。

> [!figure]+ 甲类、乙类和丙类放大器波形 Page 16
> ![[Screenshot_20250419_211458.jpg|400]]
* 但 $\theta_c$ 过小会导致输出功率 $P_o$ 下降，除非大幅提高激励和工作电压。
* 丁类（及戊类）放大器采用**开关工作状态**，使晶体管工作在饱和区（导通， $V_{CE} \approx V_{CES}$ 很小）或截止区（截止， $I_C \approx 0$），从而**极大地减小管耗** $P_c = V_{CE} \times I_C$。
* 理想情况下，效率 $\eta$ 可接近 100%。

### 2. 电压开关型D类功放原理

* **电路结构**: 通常由两个开关管（如 VT1, VT2）和输出串联谐振回路（L, C, $R_L$）组成。输入信号 $u_i$ 经变压器产生两个反相的激励信号 $u_{b1}, u_{b2}$ 分别驱动两个管子。

>[!figure]+ 基本电压开关型D类功放电路 Page 19
>![[Screenshot_20250419_212239.jpg|300]]
* **工作过程**:
    * $u_i$ 正半周: VT1 饱和导通 ($V_{CE1} \approx V_{CES}$), VT2 截止。A 点电压 $u_A \approx V_{CC} - V_{CES}$。
    * $u_i$ 负半周: VT2 饱和导通 ($V_{CE2} \approx V_{CES}$), VT1 截止。A 点电压 $u_A \approx V_{CES} \approx 0$ （假设 VT2 的电源由 C 提供）。
    * A 点产生近似幅值为 $V_{CC} - 2V_{CES}$ 的矩形波电压 $u_A$。
* **输出**: $LC R_L$ 串联谐振回路调谐在输入信号基波频率 $\omega$ 上，且 Q 值足够高。该回路对 $u_A$ 的基波分量呈现纯阻性，对高次谐波呈现很高的阻抗。因此，流过负载 $R_L$ 的电流 $i_{c1}$ 或 $i_{c2}$ 近似为基波正弦（或余弦）波，输出功率 $P_o$ 集中在基波。

>![[Screenshot_20250419_212956.jpg|350]]

### 3. 输出功率与效率

* $u_A$ (近似方波) 的基波幅值 $U_{A1m} \approx \frac{4}{\pi} V_{CC}$ (忽略 $V_{CES}$)。
* 输出功率 $$P_o = \frac{1}{2} \frac{U_{A1m}^2}{R_L} = \frac{1}{2} (\frac{2V_{CC}}{\pi})^2 \frac{1}{R_L} = \frac{8 V_{CC}^2}{\pi^2 R_L}$$
* 电源提供的直流功率 $$P_D = V_{CC} \times (I_{D1} + I_{D2})$$每个管子的平均电流 $$I_D = \frac{1}{T} \int_0^{T/2} i_{c1} dt = \frac{1}{T} \int_0^{T/2} \frac{U_{A1m}}{R_L} \sin \omega t dt = \frac{ U_{A1m}}{\pi R_L} = \frac{2 V_{CC}}{\pi^2 R_L}$$
* 总直流功率 $$P_D = 2 V_{CC} I_D = \frac{8 V_{CC}^2}{\pi^2 R_L}$$
* 理想效率 $\eta = \frac{P_o}{P_D} = 100\%$。
* 实际效率小于 100% (通常 > 90%)，原因包括：$V_{CES}$ 不为零、开关管的开关时间（上升下降沿）损耗、二极管恢复损耗（如果使用）、谐振回路损耗、寄生参数影响等。

## 5.8 宽带高频功率放大器

### 1. 概述

* **窄带功放**: 以 LC 谐振回路为输出电路，相对通频带 B/f₀ 很窄（百分之几到千分之几），并且需要调谐，所以又称为窄带高频功率放大器。由于调谐系统复杂，窄带功率放大器的运用就受到了很大的限制 。
* **宽带功放**: 能在很宽的频带内（如几个倍频程）实现不调谐工作。
    * 采用**非调谐**单元作为输出匹配电路，以频率特性很宽的**传输线变压器 (Transmission Line Transformer, TLT)**代替了电阻电容活电感作为其输出电路。
    * **缺点**: 没有选频作用，谐波抑制是重要问题。通常工作在非线性畸变比较小的甲类或甲乙类，非线性失真小，但效率相对较低。以牺牲效率换取宽频带输出。

### 2. 普通变压器的宽带限制

* **等效电路**: 包含绕组电感 $L_p$、漏感 $L_s$、损耗电阻 $r$、分布电容 $C$。
* **低频限制**: $L_p$ 的感抗 $\omega L_p$ 随频率降低而减小，对信号源产生分流作用，导致低频响应下降。
* **高频限制**: 漏感 $L_s$ 和分布电容 $C$ 形成串联或并联谐振，导致在高频端出现谐振峰，偏离谐振点后响应急剧下降。
> [!figure]+ 普通变压器等效电路
> ![[Screenshot_20250419_220128.jpg|400]]
* **改善措施**:
    * 减小 $L_s$ 和分布电容 $C$ (铁氧体磁芯、少匝数、匝间大间距)。
    * 减小磁芯损耗 (高频铁氧体)。
    * 增大 $L_p$ 以改善低频响应 (高导磁率磁芯、大截面积)。

### 3. 传输线变压器 (TLT) 

* **结构**: 两根（或多根）等长导线紧密绞合或平行排布，一同绕在高导磁率磁芯上。
* **原理**:
    * 能量主要通过导线间的**传输线模式 (TEM波)** 传播，而不是依靠磁芯的磁通耦合（虽然磁芯用于抑制共模电流，提高低频性能）。
    * 紧密耦合使得线间分布电容大，同时绕在磁芯上使得分布（共模）电感大。
    * 分布电容成为传输线特性阻抗的一部分，不限制高频响应，电磁波以电磁能交换的形式在导线间介质中传播。
    * 工作频率可达几百 MHz 甚至 GHz。

### 4. 常用 TLT 分析

* **1:1 TLT (倒相器/Balun)**: [source: 30]
    * 实现信号倒相或平衡-不平衡转换。
    * 输入阻抗 $Z_{in} = Z_{out} = R_L$。
    * 实现信号源与负载的最佳匹配要求 $R_S = R_L$，且传输线特性阻抗 $Z_0 = \sqrt{R_S R_L} = R_S = R_L$。[Image: 1:1 TLT 电路 Page 26]
* **1:4 TLT (阻抗变换)**: [source: 31]
    * 通过特定连接方式，使得输入电压 $u_{in} = u$, 输入电流 $i_{in} = 2i$, 输出电压 $u_{out} = 2u$, 输出电流 $i_{out} = i$。
    * 输入阻抗 $Z_{in} = \frac{u_{in}}{i_{in}} = \frac{u}{2i} = \frac{u}{2(u_{out}/R_L)} = \frac{u}{2(2u/R_L)} = \frac{R_L}{4}$。
    * 实现将负载阻抗 $R_L$ **降低** 4 倍。[Image: 1:4 TLT 电路连接 Page 27]
* **4:1 TLT (阻抗变换)**: [source: 31]
    * 将 1:4 TLT 的输入输出端对调。
    * 实现将负载阻抗 $R_L$ **升高** 4 倍。[Image: 4:1 TLT 电路连接 Page 27]

## 5.10 晶体管倍频器 [source: 32]

### 1. 功用与类型

* **功用**: 产生频率为输入信号频率整数倍 ($n f_{in}$) 的输出信号。
* **目的**: 在发射机中，可以使用稳定度较高的低频主振荡器，然后通过倍频获得所需的发射频率。
* **类型**:
    * **丙类倍频器**: 利用丙类放大器集电极电流脉冲中丰富的谐波成分。
    * **参量倍频器**: 利用 PN 结电容的非线性（变容二极管）。

### 2. 丙类倍频器 [source: 33]

* **电路结构**: 与丙类谐振功放基本相同，但**输出谐振回路调谐在所需的 n 次谐波频率 $n \omega$ 上**，而不是基波 $\omega$。[Image: 丙类倍频器电路 Page 29]
* **工作原理**:
    * 丙类放大器的集电极电流 $i_c$ 是尖顶余弦脉冲，包含丰富的谐波分量: $i_c = I_{c0} + I_{c1}\cos\omega t + ... + I_{cn}\cos n\omega t + ...$
    * 输出谐振回路对 $n\omega$ 谐波呈现高阻抗（谐振阻抗 $R_p$），对基波和其他次谐波呈现低阻抗。
    * 因此，在负载上主要得到 $n\omega$ 频率的电压和功率输出。

### 3. 工作特点与分析 [source: 34, 35]

* **集电极电压**: 在 $i_c$ 脉冲期间，$u_{ce}$ 下降；但在 $i_c$ 截止期间，由于回路中是 $n\omega$ 的振荡， $u_{ce}$ 的变化速率更快，可能导致 $u_{ce}$ 波形在 $i_c$ 导通期间也较高，增大了管耗 $P_c$。[Image: 二次倍频波形示意 Page 30]
* **效率**: 倍频器的效率 $\eta_c$ 低于基波放大器，且倍频次数 $n$ 越高，效率越低。因此通常只用于二倍频、三倍频。
* **最佳导通角**: 为获得最大的 n 次谐波电流分量 $I_{cn}$，理论上存在一个最佳导通角 $\theta_c$。经验公式为 $\theta_c \approx \frac{120^\circ}{n}$。
    * $n=2$ (二倍频): $\theta_c \approx 60^\circ$
    * $n=3$ (三倍频): $\theta_c \approx 40^\circ$
* **输出功率与效率公式**:
    * 输出功率 $P_{on} = \frac{1}{2} U_{cn} I_{cn} = \frac{1}{2} (\xi V_{CC}) I_{cmax} \alpha_n(\theta_c)$
    * 效率 $\eta_c = \frac{P_{on}}{P_D} = \frac{1}{2} \xi g_n(\theta_c)$, 其中 $g_n(\theta_c) = \frac{I_{cn}}{I_{c0}} = \frac{\alpha_n(\theta_c)}{\alpha_0(\theta_c)}$。
    * $P_{on}$ 正比于 n 次谐波分解系数 $\alpha_n(\theta_c)$。
