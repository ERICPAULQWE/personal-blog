# 1.1 连续时间和离散时间信号
## 1.1.1 数学表示
连续时间信号：$x(t)$
离散时间信号：$x[n]$  $n$为整数

## 1.1.2 信号能量与功率
### 信号的能量
**连续时间信号总能量**($t_1\le t \le t_2$)：$E=\int^{t_2}_{t_1}x|(t)|^2\ dt$
**离散时间信号总能量**($n_1\le n \le n_2$)：$E=\sum^{n_2}_{n=n_1}x|(n)|^2$
### 信号的平均功率
**连续时间信号平均功率**(无穷区间)：$P_{\infty}=\lim_{T \rightarrow \infty}\frac{1}{2T}\int^{T}_{-T}x|(t)|^2\ dt$
**离散时间信号平均功率**(无穷区间)：$P_{\infty}=\lim_{N \rightarrow \infty}\frac{1}{2N+1}\sum^{N}_{-N}x|(n)|^2$
### 信号的分类（按能量和功率）
**第一类信号**(能量有限信号)：总能量有限，无穷区间平均功率为0
**第二类信号**(功率有限信号)：总能量无限，无穷区间平均功率有限
**第三类信号**(功率无限信号)：总能量无限，无穷区间平均功率无限

# 1.2 自变量的变换
## 1.2.1 自变量变换

| 变换类型   | 连续信号       | 离散信号       |
| ------ | ---------- | ---------- |
| 时移     | $x(t-t_0)$ | $x[n-n_0]$ |
| 时间反转   | $x(-t)$    | $x[-n]$    |
| 时间尺度变换 | $x(at)$    | $x[an]$    |

## 1.2.2 周期信号
连续时间周期信号：$x(t)=x(t+T)$
离散时间周期信号：$x[n]=x[n+N]$
**基波周期**：最小的周期(使得上式成立的最小正值$T,N$)
## 1.2.3 奇信号和偶信号
偶信号：$x(-t)=x(t)$
奇信号：$x(-t)=-x(t)$
任何信号都能分解为一个奇信号和一个偶信号之和：
$$x(t)=Even(t)+Odd(t)$$
$$Even(t)=\frac{x(t)+x(-t)}{2}$$
$$Odd(t)=\frac{x(t)-x(-t)}{2}$$
