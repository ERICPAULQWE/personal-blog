* 工程中存在大量非周期信号。

## 4.1 非周期信号的表示——连续时间傅立叶变换

### 一、从傅立叶级数到傅立叶变换

* **思路**：将非周期信号视为周期信号在周期 $T_0$ 趋于无穷大 ($T_0 \to \infty$) 时的极限情况 [s。
* $\tilde{x}(t)$为周期信号，$x(t)$为非周期信号。
* **周期信号傅立叶级数**：
    $\tilde{x}(t)=\sum_{k=-\infty}^{\infty}a_{k}e^{jk\omega_{0}t}$，其中 $\omega_0 = \frac {2\pi} {T_0}$ 是基波角频率，傅立叶系数$a_k=\frac{2T_1 \sin k \omega_0 T_1}{T_0k\omega_0 T_1 }$。
* **周期矩形脉冲示例** [source 4-5]：
    * 当周期 $T_0$ 增大时：
        * 频谱幅度 $|a_k|$ 下降 (与 $1/T_0$ 成正比)。
        * 谱线间隔 $\omega_0$ 减小。
        * 频谱包络形状不变。
>[!figure]+ Image: 周期矩形脉冲信号及其频谱 ($T_0=4T_1$ 和 $T_0=8T_1$) 
>![[Screenshot_20250424_162741.jpg|400]]
* **极限情况 ($T_0 \to \infty$)**：
    * $\omega_0 \to d\omega$ （基频趋于无穷小）。
    * $k\omega_0 \to \omega$ （离散频率变为连续频率）。
    * $a_k \to 0$ （各频率分量幅度趋于零）。
    * 离散谱演变为幅度无限小的连续谱。
* **引入频谱密度函数**：
    * 虽然 $a_k \to 0$，但 $T_0 a_k$ 在 $T_0 \to \infty$ 时是有限值$T_0a_k=2T_1\frac{\sin k \omega_0 T_1}{k \omega_0T_1}$。
    * 定义 **频谱密度函数** $X(j\omega) = \lim_{T_0 \to \infty} T_0 a_k$。
    * $$\boxed{X(j\omega) = \int_{-\infty}^{\infty}x(t)e^{-j\omega t}dt}$$ —— **连续时间傅立叶变换 (CTFT)**。
    * $X(j\omega)$ 代表单位频带上的频谱值。
* **傅立叶系数与变换的关系**：
    * $$\boxed{a_k = \frac{1}{T_0} \int_{-T_0/2}^{T_0/2} \tilde{x}(t) e^{-jk\omega_0 t} dt = \frac{1}{T_0} X(jk\omega_0)}$$
    * 表明周期信号的频谱 $a_k$ 是对应非周期信号频谱 $X(j\omega)$ 在 $k\omega_0$ 处的 **采样** 值，乘以 $1/T_0$。
* **傅立叶反变换/逆变换 (ICTFT)**：
    * 从傅立叶级数 $\tilde{x}(t) = \sum_{k=-\infty}^{\infty} a_k e^{jk\omega_0 t}$ 出发，代入 $a_k = \frac{1}{T_0} X(jk\omega_0)$ 并取 $T_0 \to \infty$ 的极限。
    * $$\boxed{x(t) = \frac{1}{2\pi} \int_{-\infty}^{\infty} X(j\omega) e^{j\omega t} d\omega}$$
    * 表明非周期信号可以分解为频率连续分布、幅度为 $\frac{1}{2\pi}X(j\omega)d\omega$ 的无穷多个复指数信号之和。
* **连续时间傅立叶变换对** ：对非周期信号的频域描述方法
    * $$\boxed{X(j\omega)=\int_{-\infty}^{\infty}x(t)e^{-j\omega t}dt}$$
    * $$\boxed{x(t)=\frac{1}{2\pi}\int_{-\infty}^{\infty}X(j\omega)e^{j\omega t}d\omega}$$
* **周期信号与非周期信号频谱关系总结**：
    * 周期信号频谱是对应非周期信号频谱的**样本**。
    * 非周期信号频谱是对应周期信号频谱的**包络**。

### 二、傅立叶变换的收敛条件

* 存在两组充分条件（满足其一即可）：
    1.  **平方可积条件**：信号能量 $\int_{-\infty}^{\infty}|x(t)|^{2}dt < \infty$ 。
    2.  **Dirichlet 条件**：
        * a. 信号绝对可积： $\int_{-\infty}^{\infty}|x(t)|dt < \infty$。
        * b. 在任何有限区间内，$x(t)$ 只有有限个极值点，且极值有限。
        * c. 在任何有限区间内，$x(t)$ 只有有限个第一类间断点（跳跃间断点）。
* **注意**：这些仅是 **充分条件**，并非必要条件。例如 $x(t) = \frac{\sin(t)}{t}$ 平方可积但并非绝对可积，其傅立叶变换仍然存在。
* **收敛性**：
    * 在 $x(t)$ 的连续点处，其傅立叶变换变换收敛于 $x(t)$ 本身。
    * 在 $x(t)$ 的间断点处，其傅立叶变换收敛于该点左右极限的**平均值**。在间断点附近可能出现**吉布斯现象**。

### 三、常用信号的傅立叶变换

1.  **单边指数衰减信号**： $x(t)=e^{-at}u(t)$, $a>0$
    * $X(j\omega) =  \int_{0}^{\infty} e^{at}e^{-j\omega t}\,dt=\frac{1}{a+j\omega}$
    * $|X(j\omega)| =\frac{1}{\sqrt{a^2 + \omega^2}}$
    * $\angle X(j\omega) = -\arctan(\frac{\omega}{a})$

2.  **双边指数衰减信号**： $x(t)=e^{-a|t|}$, $a>0$
    * $X(j\omega) =  \int_{-\infty}^{0} e^{at}e^{-j\omega t}\,dt+\int_{0}^{\infty} e^{-at}e^{-j\omega t}\,dt=\frac{2a}{a^2 + \omega^2}$
    * $|X(j\omega)|=X(j\omega)$
    * $\angle X(j\omega)=0$
    * 此信号为实偶信号，其傅立叶变换为**实偶函数**，相位恒为0。

3.  **单位冲激信号**： $x(t)=\delta(t)$
    * $X(j\omega) = \int_{-\infty}^{\infty}\delta(t)e^{-j\omega t}dt = 1$
    * 表明 $\delta(t)$ 包含了所有频率成分，且幅度均为1，相位均为0，因此系统的单位冲激响应$h(t)$才能完全描述一个LTI系统的特性。这是 $\delta(t)$ 在系统分析中重要的原因。
    * $\delta(t)信号频谱$

4.  **矩形脉冲信号**： $x(t) = \begin{cases} 1, & |t| < T_1 \\ 0, & |t| > T_1 \end{cases}$
    * $X(j\omega) = \int_{-T_1}^{T_1} e^{-j\omega t} dt = \frac{2 \sin(\omega T_1)}{\omega} = 2T_1 Sa(\omega T_1) = 2T_1 \text{sinc}(\frac{\omega T_1}{\pi})$
        * 其中 $Sa(x) = \frac{\sin(x)}{x}$ 称为抽样函数。
        * $\text{sinc}(x) = \frac{\sin(\pi x)}{\pi x}$ 是归一化sinc函数。
        * 显然将$X(j\omega)$中的$\omega$代之以$k\omega_0$再乘以$\frac{1}{T_0}$，即是相应周期信号的频谱$$a_k=\frac{1}{T_0}X(jk\omega_0)=\frac{2T_1}{T_0}Sa(k\omega_0T_1)=\frac{2T_1}{T_0}\frac{\sin k \omega_0T_1}{k\omega_0T_1}$$
    * **脉冲宽度与频谱关系**：
        * 时域脉冲越**窄**，频域主瓣越**宽**。
        * 时域脉冲越**宽**，频域主瓣越**窄**。
        * 这体现了时域和频域的**测不准原理**或**相反关系 (inverse relationship)**。

5.  **理想低通滤波器 (频域矩形脉冲)**： $X(j\omega) = \begin{cases} 1, & |\omega| < W \\ 0, & |\omega| > W \end{cases}$
    * $x(t) = \frac{1}{2\pi} \int_{-W}^{W} e^{j\omega t} d\omega = \frac{\sin(Wt)}{\pi t} = \frac{W}{\pi} Sa(Wt) = \frac{W}{\pi} \text{sinc}(\frac{Wt}{\pi})$

6.  **直流信号 (常数)**： $x(t)=1$
    * $X(j\omega) = 2\pi\delta(\omega)$
    * 推导：由 $x(t) = \frac{1}{2\pi} \int_{-\infty}^{\infty} X(j\omega) e^{j\omega t} d\omega$，若 $X(j\omega) = 2\pi\delta(\omega)$，则 $x(t) = \frac{1}{2\pi} \int_{-\infty}^{\infty} 2\pi\delta(\omega) e^{j\omega t} d\omega = e^{j0t} = 1$。

### 四、信号的带宽 (Bandwidth of Signals)

* 信号能量通常集中在低频部分。
* 工程上传输信号时，只需传输包含主要能量的频率分量。
* **带宽定义方法**：
    1.  **3dB 带宽**：幅度谱 $|X(j\omega)|$ 下降到最大值的 $1/\sqrt{2}$ (约0.707) 时对应的频率范围。此时带内信号分量约占信号总能量的一半。
    2.  **主瓣宽度**：对于 $Sa(x)$ 或类似形状的频谱，常定义第一个零点频率内的范围为主瓣宽度，作为信号带宽。
    3.  **脉宽带宽积**：对于矩形脉冲等信号，脉冲宽度 $\times$ 带宽 $\approx$ 常数。这再次印证了时域和频域的相反关系。

## 4.2 周期信号的傅立叶变换

* **目标**：统一周期信号和非周期信号的频域表示方法。
* **问题**：周期信号通常不满足绝对可积条件，不能直接用 $\int_{-\infty}^{\infty}x(t)e^{-j\omega t}dt$ 定义。
* **利用冲激函数**：
    * 考虑 $X(j\omega) = 2\pi\delta(\omega - \omega_0)$，其反变换为 $x(t) = \frac{1}{2\pi} \int_{-\infty}^{\infty} 2\pi\delta(\omega - \omega_0) e^{j\omega t} d\omega = e^{j\omega_0 t}$。
    * 这表明单个复指数信号 $e^{j\omega_0 t}$ 的傅立叶变换是在 $\omega = \omega_0$ 处的一个**冲激**，强度为 $2\pi$。
* **周期信号的傅立叶变换**：
    * 将周期信号 $x(t)$ 展开为傅立叶级数： $x(t) = \sum_{k=-\infty}^{\infty} a_k e^{jk\omega_0 t}$。
    * 利用线性和 $e^{jk\omega_0 t} \leftrightarrow 2\pi\delta(\omega - k\omega_0)$ 的关系：
    * $X(j\omega) = \mathcal{F} \{ \sum_{k=-\infty}^{\infty} a_k e^{jk\omega_0 t} \} = \sum_{k=-\infty}^{\infty} a_k \mathcal{F} \{ e^{jk\omega_0 t} \} = \sum_{k=-\infty}^{\infty} a_k [2\pi\delta(\omega - k\omega_0)]$
    * $$\boxed{X(j\omega) = 2\pi \sum_{k=-\infty}^{\infty} a_k \delta(\omega - k\omega_0)}$$
    * **结论**：周期信号的傅立叶变换是在各次谐波频率 $k\omega_0$ 处的一系列 **冲激**，每个冲激的强度 (面积) 等于 $2\pi$ 乘以对应的傅立叶系数 $a_k$。
* **例子**：
    1.  $x(t) = \sin(\omega_0 t) = \frac{1}{2j}(e^{j\omega_0 t} - e^{-j\omega_0 t})$
        * $a_1 = \frac{1}{2j}$, $a_{-1} = -\frac{1}{2j}$
        * $X(j\omega) = 2\pi [ \frac{1}{2j} \delta(\omega - \omega_0) - \frac{1}{2j} \delta(\omega + \omega_0) ] = \frac{\pi}{j} [\delta(\omega - \omega_0) - \delta(\omega + \omega_0)]$
    2.  $x(t) = \cos(\omega_0 t) = \frac{1}{2}(e^{j\omega_0 t} + e^{-j\omega_0 t})$
        * $a_1 = a_{-1} = \frac{1}{2}$
        * $X(j\omega) = 2\pi [ \frac{1}{2} \delta(\omega - \omega_0) + \frac{1}{2} \delta(\omega + \omega_0) ] = \pi [\delta(\omega - \omega_0) + \delta(\omega + \omega_0)]$
    3.  **均匀冲激串** $x(t) = \sum_{n=-\infty}^{\infty} \delta(t-nT)$
        * 周期为 $T$，基频 $\omega_0 = 2\pi/T$。
        * 傅立叶系数 $a_k = \frac{1}{T} \int_{-T/2}^{T/2} \delta(t) e^{-jk\omega_0 t} dt = \frac{1}{T}$ (所有系数相等)。
        * $X(j\omega) = 2\pi \sum_{k=-\infty}^{\infty} \frac{1}{T} \delta(\omega - k\frac{2\pi}{T}) = \frac{2\pi}{T} \sum_{k=-\infty}^{\infty} \delta(\omega - k\omega_0)$
        * 结论：时域冲激串的傅立叶变换还是频域冲激串。
        * 时域均匀冲激串及其频域冲激串
        * ![[Screenshot_20250425_081703.jpg|400]]
    4.  **周期矩形脉冲**
        * 傅立叶系数 $a_k = \frac{2T_1}{T_0} Sa(k\omega_0 T_1)$。
        * $X(j\omega) = 2\pi \sum_{k=-\infty}^{\infty} [\frac{2T_1}{T_0} Sa(k\omega_0 T_1)] \delta(\omega - k\omega_0)$。
        * 周期矩形脉冲信号

## 4.3 连续时间傅立叶变换的性质

讨论性质的目的：揭示时域特性与频域特性的关系，简化变换计算。

1.  **线性 (Linearity)**：
    * 若 $x(t) \leftrightarrow X(j\omega)$，$y(t) \leftrightarrow Y(j\omega)$
    * 则 $\boxed{ax(t) + by(t) \leftrightarrow aX(j\omega) + bY(j\omega)}$
2.  **时移 (Time Shifting)**：
    * 若 $x(t) \leftrightarrow X(j\omega)$
    * 则 $\boxed{x(t-t_0) \leftrightarrow X(j\omega)e^{-j\omega t_0}}$
    * 时移不改变幅度谱 $|X(j\omega)|$，仅引入线性相位移 $-\omega t_0$。
3.  **反转 (Time Reversal)**：
    * 若 $x(t) \leftrightarrow X(j\omega)$
    * 则 $x(-t) \leftrightarrow X(-j\omega)$
4.  **尺度变换 (Scaling)**：
    * 若 $x(t) \leftrightarrow X(j\omega)$
    * 则 $\boxed{x(at) \leftrightarrow \frac{1}{|a|} X(j\frac{\omega}{a})}$
    * 时域 **压缩** ($|a|>1$) 对应频域 **扩展** 和幅度 **缩小**。
    * 时域 **扩展** ($|a|<1$) 对应频域 **压缩** 和幅度 **放大**。
    * 再次验证了时域和频域的 **相反关系** 和 **脉宽带宽积** 的概念。
5.  **共轭对称性 (Conjugate and Symmetry)**：
    * 若 $x(t) \leftrightarrow X(j\omega)$
    * 则 $\boxed{x^*(t) \leftrightarrow X^*(-j\omega)}$
    * **对于实信号 $x(t)$ ($x(t) = x^*(t)$)**：
        * $X(j\omega) = X^*(-j\omega)$ (共轭对称)
        * 实部 $Re[X(j\omega)]$ 是 $\omega$ 的 **偶函数**
        * 虚部 $Im[X(j\omega)]$ 是 $\omega$ 的 **奇函数**
        * 幅度 $|X(j\omega)|$ 是 $\omega$ 的 **偶函数**
        * 相位 $\angle X(j\omega)$ 是 $\omega$ 的 **奇函数**
    * **对于实偶信号 $x(t)$ ($x(t)=x(-t)$, $x(t)$ is real)**：
        * $X(j\omega)$ 是 **实偶函数** ($X(j\omega) = X(-j\omega) = X^*(j\omega)$)
    * **对于实奇信号 $x(t)$ ($x(t)=-x(-t)$, $x(t)$ is real)**：
        * $X(j\omega)$ 是 **纯虚奇函数** ($X(j\omega) = -X(-j\omega) = -X^*(j\omega)$)
    * **实信号分解**： $x(t) = x_e(t) + x_o(t)$
        * $x_e(t) \leftrightarrow Re[X(j\omega)]$
        * $x_o(t) \leftrightarrow j Im[X(j\omega)]$
    * **例：单位阶跃信号 $u(t)$ 的频谱** [source 37-38]
        * 分解 $u(t) = u_e(t) + u_o(t)$
        * 偶部 $u_e(t) = \frac{1}{2} \leftrightarrow \pi\delta(\omega)$
        * 奇部 $u_o(t) = \frac{1}{2} \text{Sgn}(t)$
        * 符号函数 $\text{Sgn}(t) \leftrightarrow \frac{2}{j\omega}$
        * 推导过程: $\text{Sgn}(t) = \lim_{a \to 0} [e^{-at}u(t) - e^{at}u(-t)]$$$\begin{align*}
\mathbb{F}[\text{Sgn}(t)] &= \lim_{a \to 0} \left[ \frac{1}{a + j\omega} - \frac{1}{a - j\omega} \right] \\
&= \lim_{a \to 0} \frac{-j2\omega}{a^2 + \omega^2} = \frac{2}{j\omega}
\end{align*}$$
        * 所以 $u_o(t) \leftrightarrow \frac{1}{2} \cdot \frac{2}{j\omega} = \frac{1}{j\omega}$
        * 最终 $u(t) \leftrightarrow \pi\delta(\omega) + \frac{1}{j\omega}$
        * 单位阶跃信号及其奇偶分解
        * ![[Screenshot_20250425_084847.jpg|150]]![[Screenshot_20250425_084912.jpg|150]]
6.  **时域微分与积分 (Differentiation and Integration)**：
    * **微分**： $\boxed{\frac{dx(t)}{dt} \leftrightarrow j\omega X(j\omega)}$ （微分运算变乘法）
    * **积分**： $\boxed{\int_{-\infty}^{t}x(\tau)d\tau \leftrightarrow \frac{1}{j\omega}X(j\omega) + \pi X(0)\delta(\omega)}$
        * $X(0) = \int_{-\infty}^{\infty} x(t) dt$ 是信号的直流分量（或面积）。
        * 积分项 $\pi X(0)\delta(\omega)$ 表示积分可能产生的直流分量在频域对应 $\omega=0$ 处的冲激。
        * 从 $\delta(t) \leftrightarrow 1$，利用积分性质也可得到 $u(t) \leftrightarrow \frac{1}{j\omega} + \pi \cdot 1 \cdot \delta(\omega)$。
7.  **对偶性 (Duality)** [source 40-45]：
    * 若 $x(t) \leftrightarrow X(j\omega)$
    * 则 $\boxed{X(jt) \leftrightarrow 2\pi x(-\omega)}$
    * **证明**：可以通过变换定义式或反变换式互换 $t$ 和 $\omega$ 变量得到 [source 40, 42]。
    * **应用**：可以将已知的变换对或性质通过对偶关系得到新的变换对或性质。
    * [Image: 矩形脉冲与Sa函数的对偶关系图 Page 41]
    * **由对偶性推导其他性质**：
        * **移频特性 (Frequency Shifting)** ： $x(t)e^{j\omega_0 t} \leftrightarrow X(j(\omega - \omega_0))$ （时域乘以复指数等于频域搬移）
        * **频域微分特性** [source 43-44]： $-jtx(t) \leftrightarrow \frac{d}{d\omega} X(j\omega)$
        * **频域积分特性** ： $\frac{x(t)}{-jt} + \pi x(0)\delta(t) \leftrightarrow \int_{-\infty}^{\omega} X(j\tau) d\tau$ (注：教材此公式形式可能需注意使用条件)
8.  **帕塞瓦尔定理 (Parseval's Theorem)** ：
    * 若 $x(t) \leftrightarrow X(j\omega)$
    * 则 $\boxed{\int_{-\infty}^{\infty} |x(t)|^2 dt = \frac{1}{2\pi} \int_{-\infty}^{\infty} |X(j\omega)|^2 d\omega}$
    * 物理意义：信号的总**能量**在时域计算和在频域计算相等。
    * $|X(j\omega)|^2$ 被称为**能量谱密度 (Energy Spectral Density)**，表示信号能量在不同频率上的分布密度。$\frac{1}{2\pi}|X(j\omega)|^2$ 是以 $\omega$ (rad/s) 为单位的能量谱密度。

## 4.4 卷积性质 (The Convolution Property) [source 47-50]

### 一、卷积特性 [source 47-48]

* 若 $x(t) \leftrightarrow X(j\omega)$, $h(t) \leftrightarrow H(j\omega)$
* 则 $y(t) = x(t) * h(t) \leftrightarrow Y(j\omega) = X(j\omega) H(j\omega)$
* **核心思想**：==时域卷积对应频域**乘积**。==
* **物理解释**：
    * $x(t)$ 可以分解为无穷多 $e^{j\omega t}$ 的线性组合 (ICTFT)。
    * 复指数信号 $e^{j\omega t}$ 是 **LTI系统** 的 **特征函数**，即输入 $e^{j\omega t}$ 时，输出为 $H(j\omega) e^{j\omega t}$。
    * $H(j\omega) = \int_{-\infty}^{\infty} h(t) e^{-j\omega t} dt$ 是系统冲激响应 $h(t)$ 的傅立叶变换，称为系统的 **频率响应 (Frequency Response)**，是对应频率 $\omega$ 的**特征值**。
    * 因此，系统总输出 $y(t)$ 的频谱 $Y(j\omega)$ 就是输入频谱 $X(j\omega)$ 与系统频率响应 $H(j\omega)$ 的乘积。
* **频率响应 $H(j\omega)$** ：
    * 描述了LTI系统对不同频率正弦输入的**幅度和相位**的影响。
    * $|H(j\omega)|$：幅频响应。
    * $\angle H(j\omega)$：相频响应。
    * $H(j\omega)$ 可以完全表征一个LTI系统（通常限于稳定系统）。
    * **稳定性**保证 $\int_{-\infty}^{\infty} |h(t)| dt < \infty$，从而 $H(j\omega)$ 存在。

### 二、LTI系统的频域分析法

利用卷积特性分析LTI系统的步骤：
1.  求输入信号 $x(t)$ 的傅立叶变换 $X(j\omega)$。
2.  求系统冲激响应 $h(t)$ 的傅立叶变换，即频率响应 $H(j\omega)$。
3.  计算输出信号的频谱 $Y(j\omega) = X(j\omega) H(j\omega)$。
4.  求 $Y(j\omega)$ 的傅立叶反变换，得到时域输出 $y(t) = \mathcal{F}^{-1}[Y(j\omega)]$。

## 4.5 相乘性质 (The Multiplication Property)

* 利用**对偶性**从卷积性质推导。
* 若 $x_1(t) \leftrightarrow X_1(j\omega)$, $x_2(t) \leftrightarrow X_2(j\omega)$
* 则 $x_1(t) \cdot x_2(t) \leftrightarrow \frac{1}{2\pi} [X_1(j\omega) * X_2(j\omega)]$
* **核心思想**：==时域乘积对应频域 **卷积** (并乘以 $1/(2\pi)$)。==
* **幅度调制 (Amplitude Modulation)** ：
    * 时域相乘可视为一个信号（调制信号）控制另一个信号（载波）的幅度。
* **例1：移频性质再验证** 
    * $x_1(t) = x(t) \leftrightarrow X(j\omega)$
    * $x_2(t) = e^{j\omega_0 t} \leftrightarrow X_2(j\omega) = 2\pi \delta(\omega - \omega_0)$
    * $x(t)e^{j\omega_0 t} \leftrightarrow \frac{1}{2\pi} [X(j\omega) * 2\pi \delta(\omega - \omega_0)] = X(j\omega) * \delta(\omega - \omega_0) = X(j(\omega - \omega_0))$
* **例2：正弦幅度调制 (AM)** [source 54-56]
    * 调制信号 $s(t) \leftrightarrow S(j\omega)$
    * 载波 $p(t) = \cos(\omega_0 t) \leftrightarrow P(j\omega) = \pi[\delta(\omega - \omega_0) + \delta(\omega + \omega_0)]$
    * 已调信号 $r(t) = s(t)p(t)$
    * $R(j\omega) = \frac{1}{2\pi} [S(j\omega) * P(j\omega)] = \frac{1}{2\pi} [S(j\omega) * \pi(\delta(\omega - \omega_0) + \delta(\omega + \omega_0))]$
    * $R(j\omega) = \frac{1}{2} [S(j(\omega - \omega_0)) + S(j(\omega + \omega_0))]$
    * 结论：正弦幅度调制将调制信号的频谱 $S(j\omega)$ **搬移** 到 $\pm \omega_0$ 位置，幅度减半。
    * [Image: 正弦幅度调制过程示意 Page 54]
    * [Image: 正弦幅度调制的频谱搬移 Page 55]
* **例3：同步解调 (Synchronous Demodulation)** [source 56-57]
    * 将接收到的已调信号 $r(t)$ 再次乘以同频同相的载波 $\cos(\omega_0 t)$。
    * $r(t) \cos(\omega_0 t) = \frac{1}{2} [s(t) + s(t)\cos(2\omega_0 t)]$
    * 其频谱包含原始信号频谱 $S(j\omega)$（在基带）和搬移到 $\pm 2\omega_0$ 的频谱。
    * 使用 **理想低通滤波器 (LPF)** 滤除 $2\omega_0$ 附近的高频分量，即可恢复出 $s(t)$（或其比例）。
    * [Image: 同步解调后信号频谱 Page 56]
    * [Image: 用于解调的理想低通滤波器 Page 57]
    * 低通滤波器的截止频率 $\omega_c$ 需要满足 $\omega_M < \omega_c < 2\omega_0 - \omega_M$ （假设 $S(j\omega)$ 的带宽为 $\omega_M$）。
* **例4：中心频率可变的带通滤波器** [source 58-60]
    * 通过将信号先乘以 $e^{-j\omega_0 t}$ (频谱右移 $\omega_0$)，再通过一个固定的低通滤波器 $H_{LPF}(j\omega)$（截止频率 $\omega_c$），最后再乘以 $e^{j\omega_0 t}$ (频谱左移 $\omega_0$) 实现。
    * 等效的系统频率响应 $H(j\omega) = H_{LPF}(j(\omega - \omega_0))$。
    * 这是一个中心频率在 $\omega_0$，带宽为 $2\omega_c$ 的带通滤波器。通过改变 $\omega_0$ 可以改变滤波器的中心频率。

## 4.7 由线性常系数微分方程 (LCCDE) 表征的系统 [source 61-68]

### 一、由LCCDE描述的LTI系统的频率特性 [source 61-65]

* **一般形式**： $\sum_{k=0}^{N} a_k \frac{d^k y(t)}{dt^k} = \sum_{k=0}^{M} b_k \frac{d^k x(t)}{dt^k}$ 
* **求频率响应 $H(j\omega)$ 的方法**：
    1.  **利用特征函数**：输入 $x(t) = e^{j\omega t}$，则输出必为 $y(t) = H(j\omega)e^{j\omega t}$。代入微分方程求解 $H(j\omega)$ 。
    2.  **对方程两边取傅立叶变换** [source 63-64]：
        * 利用微分性质 $\frac{d^k y(t)}{dt^k} \leftrightarrow (j\omega)^k Y(j\omega)$ 和 $\frac{d^k x(t)}{dt^k} \leftrightarrow (j\omega)^k X(j\omega)$。
        * $\sum_{k=0}^{N} a_k (j\omega)^k Y(j\omega) = \sum_{k=0}^{M} b_k (j\omega)^k X(j\omega)$
        * $Y(j\omega) [\sum_{k=0}^{N} a_k (j\omega)^k] = X(j\omega) [\sum_{k=0}^{M} b_k (j\omega)^k]$
        * $H(j\omega) = \frac{Y(j\omega)}{X(j\omega)} = \frac{\sum_{k=0}^{M} b_k (j\omega)^k}{\sum_{k=0}^{N} a_k (j\omega)^k}$
* **结论**：由LCCDE描述的LTI系统的频率响应 $H(j\omega)$ 是一个关于 $j\omega$ 的 **有理函数** 。
* **求冲激响应 $h(t)$**：通常先求出 $H(j\omega)$，然后做傅立叶反变换 $h(t) = \mathcal{F}^{-1}[H(j\omega)]$。求有理函数的反变换常用 **部分分式展开** 法 [source 65, 67]。
* 部分分式展开法：见下面
这里的有理函数通常是指 $F(j\omega)$ 可以表示为关于变量 $j\omega$ 的两个多项式之比：
$F(j\omega) = \frac{N(j\omega)}{D(j\omega)}$
其中 $N(j\omega)$ 和 $D(j\omega)$ 分别是分子和分母多项式。

### **部分分式展开法**求傅里叶反变换的步骤：

这种方法的核心思想是将一个复杂的有理函数 $F(j\omega)$ 分解成若干个更简单的分数项之和，然后利用傅里叶变换的线性和已知的傅里叶变换对，求出每一项的反变换，最后将它们相加得到最终结果 $f(t)$。
#### 第一步：确保 $F(j\omega)$ 是真分式
部分分式展开通常要求分子多项式 $N(j\omega)$ 的阶数严格小于分母多项式 $D(j\omega)$ 的阶数。
 * 如果 $F(j\omega)$ 是 真分式（分子阶数 < 分母阶数），则可以直接进行下一步。
 * 如果 $F(j\omega)$ 是 假分式（分子阶数 ≥ 分母阶数），需要先进行多项式长除法，将其分解为一个多项式（或常数）和一个真分式之和：
   $F(j\omega) = Q(j\omega) + \frac{R(j\omega)}{D(j\omega)}$
   其中 $Q(j\omega)$ 是商多项式， $R(j\omega)$ 是余数多项式，且 $R(j\omega)$ 的阶数小于 $D(j\omega)$ 的阶数。
   $Q(j\omega)$ 部分的反变换通常涉及到冲激函数 $\delta(t)$ 及其各阶导数。例如，$\mathcal{F}^{-1}\{k\} = k\delta(t)，\mathcal{F}^{-1}\{kj\omega\} = k\delta'(t)$ 等。
   然后，我们只需要对真分式部分 $\frac{R(j\omega)}{D(j\omega)}$ 进行部分分式展开。
#### 第二步：求分母多项式 $D(j\omega)$ 的根（即极点）
找到方程 $D(j\omega) = 0$ 的所有根 $p_1, p_2, \dots, p_n$，其中 n 是分母 $D(j\omega)$ 的阶数。这些根称为 $F(j\omega)$ 的极点。根据极点的性质，部分分式展开的形式会有所不同。
#### 第三步：根据极点的类型进行部分分式展开
假设我们现在处理的是真分式 $F_r(j\omega) = \frac{R(j\omega)}{D(j\omega)}$。
##### 情况 1：单实极点 (Distinct Real Poles)
如果分母 $D(j\omega)$ 有 n 个互不相同的实数极点 $p_1, p_2, \dots, p_n$，则 $F_r(j\omega)$ 可以展开为：
$F_r(j\omega) = \frac{A_1}{j\omega - p_1} + \frac{A_2}{j\omega - p_2} + \dots + \frac{A_n}{j\omega - p_n}$
其中系数 $A_k$（也称为留数）可以通过以下公式计算：
$A_k = [(j\omega - p_k) F_r(j\omega)]_{j\omega=p_k}$
##### 情况 2：重实极点 (Repeated Real Poles)
如果分母 $D(j\omega)$ 有一个实数极点 p 重复了 m 次（即 $(j\omega - p)^m$ 是 $D(j\omega)$ 的一个因子），则展开式中对应于这个重极点的项为：
$\frac{B_1}{j\omega - p} + \frac{B_2}{(j\omega - p)^2} + \dots + \frac{B_m}{(j\omega - p)^m}$
系数 $B_k$ 可以通过以下公式计算：
$B_{m-k} = \frac{1}{k!} \frac{d^k}{d(j\omega)^k} [(j\omega - p)^m F_r(j\omega)]_{j\omega=p} \quad \text{for } k = 0, 1, \dots, m-1$
例如，$B_m = [(j\omega - p)^m F_r(j\omega)]_{j\omega=p}$   $B_{m-1} = \frac{1}{1!} \frac{d}{d(j\omega)} [(j\omega - p)^m F_r(j\omega)]_{j\omega=p}$
以此类推。
##### 情况 3：共轭复极点 (Complex Conjugate Poles)
如果 $F(j\omega)$ 的系数是实数（这在物理系统中很常见），那么复数极点总是成共轭对出现。假设有一对共轭单极点 $p = \alpha + j\beta 和 p^* = \alpha - j\beta$。展开式中对应的项为：
$\frac{C}{j\omega - p} + \frac{C^*}{j\omega - p^*}$
其中系数 $C$可以通过留数公式计算：
$C = [(j\omega - p) F_r(j\omega)]_{j\omega=p}$
另一个系数 $C^*$ 就是 $C$ 的复共轭。
如果共轭复极点是 m 重的，处理方式类似于重实极点，但计算会更复杂。
#### 第四步：求每一项的傅里叶反变换
对部分分式展开后的每一项，查找傅里叶变换表或者利用傅里叶变换的基本性质来求其反变换。这里需要注意，傅里叶反变换的结果通常与信号的因果性或稳定性假设有关。在信号与系统课程中，通常假设信号是因果稳定的，这意味着系统的所有极点都在 s 平面（或 $j\omega$ 轴对应的复平面）的左半平面 (LHP)，即 $\text{Re}(p_k) < 0$。
常用的傅里叶变换对（假设信号为因果信号，即 t<0 时 f(t)=0，常用 u(t) 表示单位阶跃函数）：
 * ##### 单极点项: $\mathcal{F}^{-1} \left\{ \frac{A}{j\omega - p} \right\}$
   如果极点 $p = -a$（即 $a = -p > 0$，极点在左半平面），则 $\frac{A}{j\omega + a}$。
   使用变换对 $\mathcal{F}\{e^{-at}u(t)\} = \frac{1}{a+j\omega}$ （其中 $a>0$）。
   那么，$\mathcal{F}^{-1} \left\{ \frac{A}{a+j\omega} \right\} = A e^{-at}u(t)$。
   或者写成极点形式，对于 LHP 中的极点 $p (\text{Re}(p) < 0)$：
   $\mathcal{F}^{-1} \left\{ \frac{A}{j\omega - p} \right\} = A e^{pt}u(t)$
 * ##### 重极点项: $\mathcal{F}^{-1} \left\{ \frac{B}{(j\omega - p)^m} \right\}$
   使用变换对 $\mathcal{F}\left\{\frac{t^{m-1}}{(m-1)!}e^{-at}u(t)\right\} = \frac{1}{(a+j\omega)^m}$ （其中 $a>0$）。
   令 $a = -p > 0$（极点 $p$ 在 LHP）。
   那么，$\mathcal{F}^{-1} \left\{ \frac{B}{(j\omega - p)^m} \right\} = B \frac{t^{m-1}}{(m-1)!} e^{pt}u(t)$
 * ##### 共轭复极点对: $\mathcal{F}^{-1} \left\{ \frac{C}{j\omega - p} + \frac{C^*}{j\omega - p^*} \right\}$
   其中 $p = \alpha + j\beta$ 且 $\alpha < 0$ （LHP 极点）。
   $f(t) = C e^{pt}u(t) + C^* e^{p^*t}u(t)$
   $f(t) = C e^{(\alpha+j\beta)t}u(t) + C^* e^{(\alpha-j\beta)t}u(t)$
   $f(t) = e^{\alpha t} (C e^{j\beta t} + C^* e^{-j\beta t}) u(t)$
   如果令 $C = |C|e^{j\theta}$，则 $C^* = |C|e^{-j\theta}$。
   $f(t) = e^{\alpha t} (|C|e^{j\theta} e^{j\beta t} + |C|e^{-j\theta} e^{-j\beta t}) u(t)$
   $f(t) = e^{\alpha t} |C| (e^{j(\beta t + \theta)} + e^{-j(\beta t + \theta)}) u(t)$
   利用欧拉公式 $e^{jx} + e^{-jx} = 2\cos(x)$：
   $f(t) = 2|C| e^{\alpha t} \cos(\beta t + \theta) u(t)$
   这个结果表示一个幅度按指数 $e^{\alpha t}$ 衰减（因为 $\alpha < 0$）的余弦振荡信号。
#### 第五步：合并结果
将所有项的反变换结果加起来，就得到最终的傅里叶反变换 $f(t)$。如果是从假分式开始的，别忘了加上 $Q(j\omega)$ 部分的反变换（通常是冲激函数及其导数）。
举个例子：
求 $F(j\omega) = \frac{j\omega + 3}{(j\omega+1)(j\omega+2)}$ 的傅里叶反变换 $f(t)$。
 * 检查真分式: 分子阶数 1，分母阶数 2，是真分式。
 * 求极点: 分母根为 $p_1 = -1, p_2 = -2$。都是单实极点，且在左半平面。
 * 部分分式展开:
   $F(j\omega) = \frac{A_1}{j\omega + 1} + \frac{A_2}{j\omega + 2}$
   计算系数：
   $A_1 = [(j\omega+1)F(j\omega)]_{j\omega=-1} = \left[\frac{j\omega + 3}{j\omega + 2}\right]_{j\omega=-1} = \frac{-1+3}{-1+2} = \frac{2}{1} = 2$
   $A_2 = [(j\omega+2)F(j\omega)]_{j\omega=-2} = \left[\frac{j\omega + 3}{j\omega + 1}\right]_{j\omega=-2} = \frac{-2+3}{-2+1} = \frac{1}{-1} = -1$
   所以，$F(j\omega) = \frac{2}{j\omega + 1} - \frac{1}{j\omega + 2}$
 * 求反变换:
   $\mathcal{F}^{-1} \left\{ \frac{2}{j\omega + 1} \right\} = 2 e^{-t}u(t)$ （这里 $a=1$）
   $\mathcal{F}^{-1} \left\{ \frac{-1}{j\omega + 2} \right\} = -1 e^{-2t}u(t)$ （这里 $a=2$）
 * 合并结果:
   $f(t) = 2e^{-t}u(t) - e^{-2t}u(t)$
这就是使用部分分式展开法求有理函数傅里叶反变换的详细步骤和方法。关键在于正确地分解有理函数，并熟练运用傅里叶变换对和性质。

### 二、频率响应的求法总结 [source 66-70]

1.  **由微分方程求 $H(j\omega)$**：直接套用有理函数公式 。
    * 例： $\frac{d^2y(t)}{dt^2} + 6 \frac{dy(t)}{dt} + 8y(t) = 2 \frac{dx(t)}{dt} + 3x(t)$ 
        * $H(j\omega) = \frac{2(j\omega) + 3}{(j\omega)^2 + 6(j\omega) + 8} = \frac{2j\omega + 3}{(j\omega+2)(j\omega+4)}$
        * 部分分式展开： $H(j\omega) = \frac{A}{j\omega+2} + \frac{B}{j\omega+4}$ (需要根据原文计算A, B值，原文档计算为 $A=-1/2$, $B=5/2$，但需核对)
        * $h(t) = (A e^{-2t} + B e^{-4t}) u(t)$ (根据常用变换对 $e^{-at}u(t) \leftrightarrow \frac{1}{a+j\omega}$)
2.  **由系统框图求 $H(j\omega)$** ：
    * 设中间变量，列写时域或频域方程，解出 $Y(j\omega)/X(j\omega)$。
    * 图中积分器对应 $\frac{1}{j\omega}$，微分器对应 $j\omega$。
3.  **互联系统的 $H(j\omega)$** [source 69-70]：
    * **级联 (Cascade)**： $H(j\omega) = H_1(j\omega) H_2(j\omega)$ 
    * **并联 (Parallel)**： $H(j\omega) = H_1(j\omega) + H_2(j\omega)$ 
    * **反馈 (Feedback)** (负反馈)： $H(j\omega) = \frac{H_1(j\omega)}{1 + H_1(j\omega) H_2(j\omega)}$ 

## 4.8 小结 (Summary) [source 71-72]

1.  连续时间傅立叶变换 (CTFT) 将连续时间信号（周期和非周期）分解为复指数信号的线性组合 。
2.  傅立叶变换的性质揭示了信号时域与频域的深刻联系 。
    * **卷积特性** 是LTI系统频域分析的基础。
    * **相乘特性** 是调制解调技术的理论基础。
3.  建立了LTI系统的频域分析方法 。
4.  对于由LCCDE或框图描述的LTI系统，可以方便地求得其频率响应 $H(j\omega)$ 。
5.  **稳定** 的LTI系统可以通过其频率响应 $H(j\omega)$ 完全描述 。
6.  建立了系统互联时总频率响应与子系统频率响应的关系 。
