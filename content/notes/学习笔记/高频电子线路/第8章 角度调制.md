## 8.1 概述

本章主要讨论角度调制，包括调频 (FM) 和调相 (PM)。

**角度调制**：使高频载波的瞬时总相角随调制信号规律变化的一种调制方式。由于频率调制和相位调制都表现为高频振荡波的总瞬时相角受到调制，故统称为角度调制。

### 基本概念

* **调频 (FM)**：使高频载波的瞬时频率按照调制信号的规律变化的一种调制方式。
* **调相 (PM)**：使高频载波的瞬时相位按照调制信号的规律变化的一种调制方式。

## 8.2 调角波的性质

### 8.2.1 瞬时频率与瞬时相位

对于一个广义的正弦波信号，其表达式可以写为：
$v(t) = V_0 \cos[\phi(t)]$
其中，$V_0$ 是振幅，$\phi(t)$ 是信号的瞬时总相角。

* **瞬时角频率** $\omega(t)$ 定义为瞬时总相角对时间的一阶导数：
    $\omega(t) = \frac{d\phi(t)}{dt}$
* **瞬时频率** $f(t)$ 定义为瞬时角频率除以 $2\pi$：
    $f(t) = \frac{1}{2\pi}\omega(t) = \frac{1}{2\pi}\frac{d\phi(t)}{dt}$
* **瞬时相位** (或瞬时总相角) $\phi(t)$ 是载波角频率 $\omega_0$ 与初始相位 $\theta_0$ 之和，再加上随调制信号变化的附加相移。
    $\phi(t) = \omega_0 t + \theta(t)$
    其中 $\theta(t)$ 是由调制信号引起的相位变化。

### 8.2.2 调频波和调相波的数学表示式

设调制信号为 $v_{\Omega}(t)$，载波信号为 $V_0 \cos(\omega_0 t + \theta_0)$。

#### 调频 (FM)

* **瞬时角频率** $\omega_{FM}(t) = \omega_0 + k_f v_{\Omega}(t)$
    * $\omega_0$ 是未调制时的载波中心角频率。
    * $k_f$ 是调频灵敏度，单位是 $\text{rad/s/V}$。
    * $k_f v_{\Omega}(t)$ 是瞬时角频率偏移，简称角频偏。
* **瞬时频率** $f_{FM}(t) = f_0 + \frac{k_f}{2\pi} v_{\Omega}(t)$。
* **最大角频偏** $\Delta\omega_m = k_f |v_{\Omega}(t)|_{\max}$。
* **最大频偏** $\Delta f_m = \frac{\Delta\omega_m}{2\pi}$。
* **瞬时相位** $\phi_{FM}(t) = \int \omega_{FM}(t) dt = \omega_0 t + k_f \int v_{\Omega}(t) dt + \theta_0$。
* **调频波表达式**: $v_{FM}(t) = V_0 \cos[\omega_0 t + k_f \int v_{\Omega}(t) dt + \theta_0]$。
* **调制指数 (单音调制时)**: 若 $v_{\Omega}(t) = V_m \cos(\Omega t)$，则最大角频偏 $\Delta\omega_m = k_f V_m$。瞬时相位中的积分项为 $\frac{k_f V_m}{\Omega} \sin(\Omega t) = \frac{\Delta\omega_m}{\Omega} \sin(\Omega t)$。
    * **调频指数** $m_f = \frac{\Delta\omega_m}{\Omega} = \frac{\Delta f_m}{F}$，其中 $F = \Omega / (2\pi)$ 是调制信号频率。
    * 已调频信号: $v_{FM}(t) = V_0 \cos[\omega_0 t + m_f \sin(\Omega t) + \theta_0]$。

#### 调相 (PM)

* **瞬时相位** $\phi_{PM}(t) = \omega_0 t + \theta_0 + k_p v_{\Omega}(t)$。
    * $k_p$ 是调相灵敏度，单位是 $\text{rad/V}$。
    * $k_p v_{\Omega}(t)$ 是瞬时相位偏移，简称相偏。
* **最大相偏** $\Delta\theta_m = k_p |v_{\Omega}(t)|_{\max}$。
* **瞬时角频率** $\omega_{PM}(t) = \frac{d\phi_{PM}(t)}{dt} = \omega_0 + k_p \frac{dv_{\Omega}(t)}{dt}$。
* **频偏**: 瞬时角频率偏移为 $k_p \frac{dv_{\Omega}(t)}{dt}$。
* **调相波表达式**: $v_{PM}(t) = V_0 \cos[\omega_0 t + k_p v_{\Omega}(t) + \theta_0]$。
* **调制指数 (单音调制时)**: 若 $v_{\Omega}(t) = V_m \cos(\Omega t)$，则最大相偏 $\Delta\theta_m = k_p V_m$。
    * **调相指数** $m_p = \Delta\theta_m = k_p V_m$。
    * 已调相信号: $v_{PM}(t) = V_0 \cos[\omega_0 t + m_p \cos(\Omega t) + \theta_0]$。
    * 瞬时角频率 $\omega_{PM}(t) = \omega_0 - k_p V_m \Omega \sin(\Omega t) = \omega_0 - m_p \Omega \sin(\Omega t)$。

#### FM 与 PM 的比较 (单音调制)

| 特性         | 调频 (FM)                                                                 | 调相 (PM)                                                                         |
| ------------ | ------------------------------------------------------------------------- | --------------------------------------------------------------------------------- |
| 调制信号     | $V_m \cos(\Omega t)$                                                 | $V_m \cos(\Omega t)$                                                         |
| 瞬时相位     | $\omega_0 t + \frac{\Delta\omega_m}{\Omega} \sin(\Omega t) + \theta_0$ | $\omega_0 t + m_p \cos(\Omega t) + \theta_0$                               |
| 瞬时角频率   | $\omega_0 + \Delta\omega_m \cos(\Omega t)$                     | $\omega_0 - m_p \Omega \sin(\Omega t)$                                     |
| 最大角频偏   | $\Delta\omega_m = k_f V_m$ (与 $\Omega$ 无关)                      | $\Delta\omega_{mp} = m_p \Omega = k_p V_m \Omega$ (与 $\Omega$ 成正比)        |
| 最大相偏     | $\Delta\theta_{mf} = m_f = \frac{\Delta\omega_m}{\Omega}$ (与 $\Omega$ 成反比) | $m_p = k_p V_m$ (与 $\Omega$ 无关)                                          |
| 调制指数     | $m_f = \frac{\Delta\omega_m}{\Omega}$                                  | $m_p = k_p V_m$                                                            |
| 表达式       | $V_0 \cos[\omega_0 t + m_f \sin(\Omega t)]$                        | $V_0 \cos[\omega_0 t + m_p \cos(\Omega t)]$                                |

可以看出，调相波的信号带宽随调制信号频率的升高而增加，而调频波则不变（指最大频偏不变，带宽与调制指数和调制频率有关）。有时把调频制叫做恒定带宽调制。

### 8.2.3 调频波和调相波的频谱和频带宽度

由于调频波和调相波的方程式相似，因此只要分析其中一种的频谱，则对另一种也完全适用。
以单音调制为例，已调频信号可以表示为：
$v_{FM}(t) = V_0 \cos[\omega_0 t + m_f \sin(\Omega t)]$
可以展开为：
$v_{FM}(t) = V_0 \sum_{n=-\infty}^{\infty} J_n(m_f) \cos(\omega_0 + n\Omega)t$
其中 $J_n(m_f)$ 是以 $m_f$ 为参数的 n 阶第一类贝塞尔函数。

#### 一、频谱特性

1.  **边带组成**：单音调制时，调频波的频谱由载波 $(\omega_0)$ 和无数对边带分量 $(\omega_0 \pm n\Omega)$ 组成。它们的振幅由对应的各阶贝塞尔函数值 $V_0 J_n(m_f)$ 确定。
    * 奇次 $(n \text{为奇数})$ 的上下边带分量振幅相等、极性相反。
    * 偶次 $(n \text{为偶数})$ 的上下边带分量振幅相等、极性相同。
2.  **调制指数影响**：调制指数 $m_f$ 越大，具有较大振幅的边频分量就越多。这与调幅波不同，在单频信号调幅的情况下，边频数目与调制指数无关。
3.  **载波与边带振幅变化**：载波分量 $(n=0)$ 和各边带分量的振幅均随 $m_f$ 变化而变化。对于某些 $m_f$ 值，载频或某边频振幅可能为零。利用这一特性可以测定调制指数 $m_f$ (例如，当 $m_f \approx 2.405$ 时，$J_0(m_f) \approx 0$，载波分量消失)。
4.  **平均功率**：根据帕塞瓦尔 (Parseval) 定理，调频波的平均功率等于各频谱分量平均功率之和。在电阻 R 上，调频波的平均功率为：
    $P_{FM} = \frac{V_0^2}{2R} \sum_{n=-\infty}^{\infty} J_n^2(m_f)$
    由于贝塞尔函数的性质 $\sum_{n=-\infty}^{\infty} J_n^2(m_f) = 1$，所以 $P_{FM} = \frac{V_0^2}{2R}$。
    这表明，当 $V_0$ 一定时，不论 $m_f$ 为何值，调频波的平均功率恒为定值，并且等于未调制时的载波功率。改变 $m_f$ 仅会引起载波分量和各边带分量之间功率的重新分配，但不会引起总功率的改变。

#### 二、带宽

理论上调频波的边频分量有无数多个，但在实际中，高到一定次数的边频分量其振幅已经小到可以忽略。
通常规定：凡是振幅小于未调制载波振幅的 1% (或 10%，根据不同要求而定) 的边频分量均可忽略不计，保留下来的频谱分量就确定了调频波的频带宽度 $B_W$。

* **卡森公式 (Carson's Rule)**：如果将小于未调制载波振幅 10% 的边频分量略去不计，则频谱宽度 $B_W$ 可由下列近似公式求出：
    $B_W \approx 2(m_f + 1)F = 2(\Delta f_m + F)$
    其中 $F$ 是调制信号的最高频率。
    更精确地，是考虑包含大约98%信号功率的带宽，这时 $B_W \approx 2(\Delta f_m + kF)$，其中 $k$ 通常取1到2。
* **窄带调频 (NBFM)**：当 $m_f \ll 1$ (例如 $m_f < 0.2$ 或 $0.5$) 时，可以近似认为只有 $J_0(m_f)$ 和 $J_1(m_f)$ 不为零。
    $v_{FM}(t) \approx V_0 [J_0(m_f)\cos(\omega_0 t) + J_1(m_f)\cos(\omega_0+\Omega)t - J_1(m_f)\cos(\omega_0-\Omega)t]$
    此时频谱类似于调幅波，带宽 $B_W \approx 2F$。
* **宽带调频 (WBFM)**：当 $m_f \gg 1$ 时，边带数量很多，带宽 $B_W \approx 2\Delta f_m = 2m_f F$。

总的规律是：调制指数越大，应当考虑的边频分量的数目就越多，无论对于调频还是调相均是如此。
但是，由于调频与调相的调制指数与调制频率 $F$ 的关系不同，当 $F$ 变化时，它们的频谱结构和频带宽度的关系就互不相同。
* **调频 (FM)**: $m_f = \Delta f_m / F$。若 $\Delta f_m$ (最大频偏) 保持不变，则 $F$ 增大时，$m_f$ 减小，有效边带数减少，但带宽仍主要由 $\Delta f_m$ 决定 (对 WBFM) 或由 $2F$ 决定 (对 NBFM)。
* **调相 (PM)**: $m_p = \Delta\theta_m$。若 $\Delta\theta_m$ (最大相偏) 保持不变，则 $m_p$ 不随 $F$ 变化。此时等效的最大频偏 $\Delta f_{mp} = m_p F$，带宽 $B_W \approx 2(m_p+1)F$ 或 $B_W \approx 2m_p F$ (对宽带)。因此，调相波的带宽随调制频率 $F$ 的增加而显著增加。

## 8.3 调频方法概述

产生调频信号的电路叫做**调频器**。主要要求：
1.  已调波的瞬时频率与调制信号成比例地变化（基本要求）。
2.  中心频率具有一定的稳定度。
3.  最大频移与调制频率无关。
4.  无寄生调幅或寄生调幅尽可能小。

产生调频信号的方法主要有两类：
1.  **直接调频**：用调制信号直接控制载波的瞬时频率。
2.  **间接调频**：由调相变调频。

### 8.3.1 直接调频原理

基本原理是用调制信号直接线性地改变载波振荡的瞬时频率。如果载波由 LC 自激振荡器产生，则振荡频率主要由谐振回路的电感 L 或电容 C 决定。只要能用调制信号去控制 L 或 C，就能达到控制振荡频率的目的。常用的方法是利用变容二极管，其结电容随反向电压变化。

### 8.3.2 间接调频原理 (阿姆斯特朗法)

由调相实现调频。基本思想是：先对调制信号进行积分，然后用积分后的信号去调相，即可得到调频波。
如果瞬时相位 $\phi(t) = \omega_0 t + k_p \int v_{\Omega}(t) dt$，则瞬时角频率 $\omega(t) = \omega_0 + k_p v_{\Omega}(t)$，这就实现了调频。
[Image 图 8.3.1 借助于调相器得到调频波 68]
![[Screenshot_20250526_212412.jpg|300]]
实现步骤：
1.  **积分器**：对调制信号 $v_{\Omega}(t)$ 进行积分。
2.  **调相器**：用积分后的信号对载波进行相位调制。
3.  **(可选) 倍频器**：为了获得较大的频偏和较高的中心频率，通常还需要使用倍频器。倍频会同时增大中心频率和频偏，但调制指数 $m_f$ 在倍频过程中保持不变（如果指的是原始信号的 $m_f$ 定义，即 $\Delta f_m/F$，那么 $\Delta f_m$ 和 $F_{carrier}$ 都倍增，但调制信号频率 $F$ 不变，因此 $m_f$ 也会倍增。这里课件的说法可能需要结合上下文理解，通常指 $m_p$ 不变，从而导致 $\Delta f_m$ 按比例增加）。

## 8.4 变容二极管调频

### 8.4.1 基本原理

变容二极管是利用半导体 PN 结的**结电容** $C_j$ 随反向电压 $v_R$ 变化这一特性而制成的一种半导体二极管。
结电容 $C_j$ 与反向电压 $v_R$ 的关系为：
$C_j = \frac{C_{j0}}{(1 + \frac{v_R}{V_D})^\gamma}$
其中：
* $C_{j0}$：$v_R=0$ 时的结电容 (零偏置电容)。
* $V_D$：PN 结的内建电势差 (势垒电位差)，通常为 0.5V ~ 0.7V。
* $v_R$：施加在 PN 结上的反向偏置电压。
* $\gamma$：结电容变化指数，通常 $\gamma = 1/2 \sim 1/3$。对于超突变结，$\gamma = 1 \sim 5$。

[Image 图 8.4.1 用调制信号控制变容二极管结电容 79]
![[20250526_212458.png|300]]
当调制信号 $v_{\Omega}(t)$ 叠加在直流反向偏置电压 $V_0$ 上，共同作用于变容二极管时，变容二极管的瞬时结电容将随 $v_{\Omega}(t)$ 的变化而变化。
$C_j(t) = C_Q + k_c v_{\Omega}(t)$ (在小信号线性近似下)
其中 $C_Q$ 是静态工作点的结电容。

* **理想线性调制条件**：要求振荡频率的变化与调制电压成严格的线性关系。
* **小频偏条件下**，可以近似实现线性调制。

**主要优点**：能够获得较大的频移（相对于间接调频而言），线路简单，并且几乎不需要调制功率。
**主要缺点**：中心频率稳定度低，因为振荡频率直接受 LC 回路参数影响，而这些参数易受温度等因素影响。
**应用范围**：在移动通信以及自动频率微调 (AFC) 系统中。

## 8.5 晶体振荡器直接调频

直接调频的主要优点是可以获得较大的频偏，但是中心频率的稳定性（主要是长期稳定性）较差。
为了提高中心频率的稳定性，可以采用对石英晶体振荡器进行直接调频。

变容二极管接入晶体振荡回路通常有两种方式：
1.  **与石英晶体相串联**。
2.  **与石英晶体相并联**。

变容二极管与晶体并联连接方式的缺点是变容管参数的不稳定性直接严重地影响调频信号中心频率的稳定度。因此用得比较广泛的是变容管与石英晶体相串联的方式。

## 8.6 间接调频: 由调相实现调频

回顾：如果瞬时相位 $\phi(t) = \omega_0 t + k_p \int v_{\Omega}(t) dt + \theta_0$，则瞬时角频率 $\omega(t) = \omega_0 + k_p v_{\Omega}(t)$。
这表明，将调制信号先积分，再用积分后的信号去进行相位调制，就可以得到频率调制信号。
这种方法通常称为**阿姆斯特朗法**。

优点：
* 中心频率的稳定性高，因为它通常由一个高稳定度的晶体振荡器决定。
缺点：
* 直接获得的频偏较小。为了得到足够大的频偏，通常需要采用多级倍频器。倍频器会同时将中心频率和频偏都乘以倍频次数 N。

## 角度解调 (鉴频与鉴相)

解调的目的是从已调波中恢复出原始的调制信号。

### 调频波的解调 (鉴频)

鉴频器的作用是把频率的变化转换成电压或电流的变化。理想的鉴频器特性是输出电压与输入信号的瞬时频率偏移成线性关系。
常见的鉴频电路有：
* 斜率鉴频器 (利用 LC 谐振回路的斜率)
* 相位鉴频器 (相位式鉴频器)
* 比例鉴频器
* 正交相乘鉴频器 (锁相环鉴频器)
---

## 8.8 相位鉴频器

#### 8.8.1 相位鉴频器的工作原理

**核心思想**：先将输入调频信号和经过相移网络后的信号进行处理（如相加或相减），产生一个其振幅能反映附加相位 $\Delta\theta$ 变化的调幅波，然后通过包络检波器检出这个振幅的变化，从而完成鉴相功能。由于调频波的瞬时相位与频率有关，因此可以间接实现鉴频。

**数学模型**:
设输入信号为 $v_1(t) = V_1 \cos[\omega_0 t + \Delta\theta(t)]$。
经过一个相移网络（例如，在中心频率 $\omega_0$ 处提供 $\pi/2$ 相移）后得到 $v_2(t)$。
例如，一个简单的模型是 $v_1(t)$ 与 $v_1(t)$ 移相后的 $v_1'(t)$ 相加：
$v_s(t) = v_1(t) + v_1'(t)$
如果 $v_1'(t)$ 的相位变化与 $v_1(t)$ 中的 $\Delta\theta(t)$ 有特定关系，则 $v_s(t)$ 的包络将与 $\Delta\theta(t)$ 相关。

[Image 图 8.8.1 相位鉴频器原理电路 104]
![[Screenshot_20250526_215810.jpg|300]]
包含一个相加器 (或减法器) 和一个包络检波器。

**实现电路**:
通常使用耦合调谐回路作为相移网络，并配合二极管包络检波器。
$L_1C_1$ 和 $L_2C_2$ 构成互感耦合的双调谐回路，作为相移网络。两个二极管包络检波器构成平衡鉴相电路。$L_3$ 为高频扼流圈， $C_0$ 为耦合电容。

**原理分析**:
输入调频信号 $v_{FM}(t) = V_m \cos[\omega_0 t + \int \Delta\omega(t)dt]$。
在中心频率 $\omega_c$ 时，相移网络（如双调谐回路）使两路信号（例如加到两个检波器的信号）有一定的相位差，例如 $\pm \pi/2$。当频率偏离 $\omega_c$ 时，这个相位差会改变，导致检波器输出电压差发生变化。

对于单个相加支路，若两信号为 $V_A \cos(\phi_A)$ 和 $V_B \cos(\phi_B)$，相加后包络的平方与 $\cos(\phi_A - \phi_B)$ 有关。
若 $V_A = V_{12}(t)$ (输入FM信号的一部分)，$V_B$ (经过相移网络的信号)。
当瞬时频率变化引起附加相位 $\Delta\theta$ 时，如果 $\phi_A - \phi_B = \frac{\pi}{2} + \Delta\theta'$, 其中 $\Delta\theta'$ 与 $\Delta\theta$ 相关。
则包络检波器输出 $V_{out}(t) \propto \sin(\Delta\theta')$。
当 $\Delta\theta'$ 较小时，$\sin(\Delta\theta') \approx \Delta\theta'$，近似线性。
但单个支路关系是非线性的，会产生失真。

**平衡鉴相电路**:
采用两个检波支路，其输出相减，可以抵消直流项和偶次谐波失真，改善线性度，并使输出信号幅值增强。
输出电压 $V_{out} \propto \sin(\Delta\theta)$。

[Image 图 8.8.4 对应于不同耦合因数的鉴频特性曲线 126]
![[20250526_220014.png|300]]
鉴频特性曲线（S曲线）的线性范围和鉴频跨导（灵敏度）与回路的耦合因数 $\eta = kQ$（$k$为耦合系数，$Q_L$为回路品质因数）有关。
* $\eta$ 很小时，线性范围小，跨导高。
* $\eta$ 增大时，线性范围变大，跨导减小。
* 通常选取 $\eta = 1 \sim 3$。

**限幅要求**：相位鉴频器对输入信号的幅度变化敏感。实际中，噪声或干扰引起的寄生调幅会直接影响输出。因此，通常在鉴频器前加**限幅器**以消除寄生调幅。

## 8.9 比例鉴频器 (Ratio Detector)

比例鉴频器的提出是为了在一定程度上克服相位鉴频器对输入信号幅度变化敏感，从而需要前级进行良好限幅的问题。它本身具有一定的**动态限幅特性**。
![[20250526_221032 1.png|300]]
**1. 工作原理**
比例鉴频器的电路结构与相位鉴频器（特别是平衡式结构）相似，主要区别在于：
* 两个检波二极管 ($D_1, D_2$) 的连接方向通常是串联的，使得它们的检波电流同向流过一个共同的负载网络。
* 负载电阻的连接方式不同，并引入一个时间常数很大的电解电容 $C_s$ (例如幻灯片中的 $C_6$)，它横跨在两个检波二极管串联组合的输出端。这个大电容的作用是稳定其两端的总电压。

输出电压 $V_o$ 通常取自两个负载电阻构成的分压网络的中点与地之间，或者是两个检波器输出电压之差的一部分。
关键在于大电容 $C_s$ 的作用。由于其时间常数很大，它两端的电压 $V_{Cs}$ (即两个检波二极管输出包络电压之和 $|V_{D1}| + |V_{D2}|$，如果二极管反向串联，则是两者之和作用于 $C_s$) 基本上保持为输入信号平均幅度的峰值或一个与之相关的稳定直流电压。
当输入信号瞬时频率变化时，$|V_{D1}|$ 和 $|V_{D2}|$ 会发生相对变化 (一个增加，一个减少)，但由于大电容的储能和稳定作用，它们的和（或一个与总幅度相关的电压）$V_{Cs}$ 基本保持不变。
输出电压 $V_o$ 与两个检波器输出电压之差 $(|V_{D1}| - |V_{D2}|)$ 成正比。
由于 $V_o \propto \frac{|V_{D1}| - |V_{D2}|}{|V_{D1}| + |V_{D2}|} \times (|V_{D1}| + |V_{D2}|)$，而 $(|V_{D1}| + |V_{D2}|)$ 被 $C_s$ 稳定，因此 $V_o$ 主要反映了频率变化，而对输入信号的幅度变化不敏感。
比例鉴频器的输出恰好等于相位鉴频器输出的一半。

**2. 动态限幅特性**
[Image 比例鉴频器动态限幅原理示意 (V12, iD, VC6, Rid, Av) 145]
![[20250526_221032.png|300]]
限幅原理：
* 当输入信号幅度 $V_{12} \uparrow$ (增大)。
* 则二极管的平均导通电流 $i_D \uparrow$ (倾向于增大)。
* 但由于大电容 $C_6$ (即 $C_s$) 两端的电压 $V_{C6}$ 不能突变，它维持了一个相对稳定的直流电压。
* 这导致二极管的等效导通电阻 $R_{id} \downarrow$ (减小) (因为回路的整体阻抗特性变化，使得二极管工作点移动)。
* 使得前级（或耦合回路）的电压放大倍数 $A_v$ (或传递到二极管的信号幅度) $A_v \downarrow$ (下降)。
* 从而抑制了输入信号幅度 $V_{12}$ 增大对输出的影响。反之，$V_{12} \downarrow$ 时亦然。
这种特性使得比例鉴频器能自动抑制一定程度的输入信号幅度波动，即具有动态限幅作用。

[Image 图 8.1.2 鉴频特性曲线 149]
比例鉴频器的鉴频特性曲线也呈 "S" 形，与相位鉴频器类似。由于其固有的限幅能力，对前级限幅器的要求降低，在某些对性能要求不非常高的场合可以省掉专用限幅器。

## 调幅、调频、调相的比较

| 特性       | 调幅 (AM)                       | 角度调制 (FM/PM)            |
| -------- | ----------------------------- | ----------------------- |
| **受控参量** | 振幅                            | 频率 (FM) 或 相位 (PM)       |
| **解调方式** | 相干解调或包络检波 (非相干解调)             | 鉴频 (FM) 或 鉴相 (PM)       |
| **频谱特点** | 线性搬移，频谱结构无变化                  | 非线性变换，频谱结构发生变化          |
| **带宽**   | 窄，通常为 $2f_m$ ($f_m$为调制信号最高频率) | 宽，由卡森公式决定，可远大于 $2f_m$   |
| **抗干扰性** | 较差                            | 强 (特别是宽带FM)             |
| **功率效率** | 较低 (载波功率占比大)                  | 较高 (恒定功率，信息主要在边带)       |
| **用途**   | 广播、电视图像信号                     | 高保真广播、电视伴音、移动通信、遥测、数字通信 |
