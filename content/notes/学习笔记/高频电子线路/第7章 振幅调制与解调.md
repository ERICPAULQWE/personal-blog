##  7.1 概述 

**调制 (Modulation)**: 将需要传输的信号（调制信号）加载到高频振荡信号（载波）上，使其某个参数（如振幅、频率、相位）随调制信号变化的过​​程。调制的主要目的是将低频的基带信号频谱搬移到高频信道中进行传输，以便于天线辐射、减少干扰、实现信道复用等。

**解调 (Demodulation/检波 Detection)**: 调制的逆过程，即从已调波中恢复出原始调制信号的过程。

**基本信号类型**:
* **调制信号 (Modulating Signal)**: 需要传输的原始信号，如语言、图像、数据等。通常是低频信号。数学上可表示为 $v_{\Omega}(t)$，例如单音频信号 $v_{\Omega}=V_{\Omega}\cos(\Omega t)$。
* **载波信号 (Carrier Signal)**: 用于承载调制信号的高频振荡信号，可以是正弦波、方波等。通常为等幅高频正弦波，数学上可表示为 $v_{c}(t)=V_{c}\cos(\omega_{c}t+\varphi)$。
* **已调信号 (Modulated Signal)**: 经过调制后的高频信号，也称射频信号。

**调制与混频**:
* 振幅调制、解调（检波）、混频（变频）都属于频谱线性搬移电路。

**调制方式分类**:
1.  **振幅调制 (Amplitude Modulation, AM)**: 载波的振幅随调制信号线性变化。
    * 普通调幅 (AM)
    * 抑制载波双边带调幅 (DSB-SC)
    * 单边带调制 (SSB)
2.  **频率调制 (Frequency Modulation, FM)**: 载波的频率随调制信号线性变化。
3.  **相位调制 (Phase Modulation, PM)**: 载波的相位随调制信号线性变化。

**解调方式分类**:
1.  **振幅检波**: 振幅调制的逆过程。
2.  **鉴频**: 调频的逆过程。
3.  **鉴相**: 调相的逆过程。

## 7.2 调幅波的性质 

### 1. 调幅波的分析

设载波信号为 $\mu_{0}=V_{0}\cos(\omega_{0}t)$，单频调制信号为 $u_{\Omega}=V_{\Omega}\cos(\Omega t)$。
标准调幅 (AM) 信号的瞬时值表达式为：
$u_{AM}(t) = V_{m}(t)\cos(\omega_{0}t)$
其中，已调波的振幅 $V_{m}(t)$ 随调制信号线性变化：
$V_{m}(t) = V_{0} + k_{a}u_{\Omega}(t) = V_{0} + k_{a}V_{\Omega}\cos(\Omega t)$
$k_{a}$ 是比例常数。
上式可以改写为：
$V_{m}(t) = V_{0}(1 + \frac{k_{a}V_{\Omega}}{V_{0}}\cos(\Omega t)) = V_{0}(1 + m_{a}\cos(\Omega t))$
其中 **调幅度 (Modulation Index)** $m_{a}$ 定义为：
$m_{a} = \frac{k_{a}V_{\Omega}}{V_{0}}$，通常用百分比表示。
因此，AM波的瞬时表达式为：
$v_{AM}(t) = V_{0}(1 + m_{a}\cos(\Omega t))\cos(\omega_{0}t)$

若调制信号为复杂的连续频谱限带信号 $f(t) = \sum_{n=1}^{\infty}V_{\Omega_{n}}\cos(\Omega_{n}t+\varphi_{n})$，则
$V_{AM}(t) = V_{0}[1+k_{a}f(t)]\cos(\omega_{0}t) = V_{0}[1+\sum_{n=1}^{\infty}m_{n}\cos(\Omega_{n}t+\varphi_{n})]\cos(\omega_{0}t)$
其中 $m_{n}=k_{a}V_{\Omega n}$。

### 2. 调幅信号波形 
[Image AM信号的调制信号、载波和已调波波形 5]

已调波的包络为 $V_{m}(t) = V_{0}(1+m_{a}\cos(\Omega t))$，其形状与调制信号 $u_{\Omega}(t)$ 一致。
包络的最大值和最小值分别为：
$V_{max} = V_{0}(1+m_{a})$
$V_{min} = V_{0}(1-m_{a})$
由此可以得到调制度的另一种计算方法：
$m_{a} = \frac{V_{max}-V_{min}}{2V_{0}} = \frac{V_{max}-V_{min}}{V_{max}+V_{min}}$ (此处 $V_0 = (V_{max}+V_{min})/2$)

**调制度 $m_a$ 的讨论**:
* $m_{a} = 0$: 未调幅，输出只有载波。
* $0 < m_{a} < 1$: 正常调幅，包络不失真。
* $m_{a} = 1$ (或 100%): 满调幅， $V_{min}=0$。这是不失真的临界状态。
* $m_{a} > 1$: 过调幅，包络出现失真（也称包络检波失真或对角切割失真），此时 $V_{min}$ 会反相，实际电路中必须避免。

### 3. 调幅波的频谱 

对于单音频调制信号 $v_{AM}(t) = V_{0}(1 + m_{a}\cos(\Omega t))\cos(\omega_{0}t)$，利用三角和差化积公式：
$\cos A \cos B = \frac{1}{2}[\cos(A-B) + \cos(A+B)]$
可得：
$v_{AM}(t) = V_{0}\cos(\omega_{0}t) + \frac{1}{2}m_{a}V_{0}\cos((\omega_{0}-\Omega)t) + \frac{1}{2}m_{a}V_{0}\cos((\omega_{0}+\Omega)t)$

调幅波包含三个频率分量：
1.  **载波分量 (Carrier)**: 频率 $\omega_{0}$，振幅 $V_{0}$。不携带调制信息。
2.  **上边频分量 (Upper Sideband, USB)**: 频率 $\omega_{0}+\Omega$，振幅 $\frac{1}{2}m_{a}V_{0}$。携带调制信息。
3.  **下边频分量 (Lower Sideband, LSB)**: 频率 $\omega_{0}-\Omega$，振幅 $\frac{1}{2}m_{a}V_{0}$。携带调制信息。

[Image 单音频AM波的频谱 6]

对于限带信号（如语音、音乐，包含从 $\Omega_{min}$ 到 $\Omega_{max}$ 的频率范围）调制的AM波：
$V_{AM}(t) =V_{0}\cos(\omega_{0}t) + \sum_{n}[\frac{1}{2}m_{n}V_{0}\cos((\omega_{0}-\Omega_{n})t) + \frac{1}{2}m_{n}V_{0}\cos((\omega_{0}+\Omega_{n})t)]$
频谱中包含：
1.  **载波分量**: $\omega_{0}$。
2.  **上边带 (USB)**: 频率范围 $(\omega_{0}+\Omega_{min})$ 到 $(\omega_{0}+\Omega_{max})$。
3.  **下边带 (LSB)**: 频率范围 $(\omega_{0}-\Omega_{max})$ 到 $(\omega_{0}-\Omega_{min})$。

[Image 限带信号AM波的频谱 8]

**结论**:
* 调幅是一种频谱线性搬移过程，调制信号的频谱被搬移到载频 $\omega_{0}$ 的两侧。
* AM波的**带宽 (Bandwidth, BW)** 为调制信号最高频率 $F_{max}$ (或 $\Omega_{max}$) 的两倍：
    $BW = 2F_{max} = 2\Omega_{max}$。

### 4. 调幅波的功率 

设调幅波加在负载电阻 $R_{L}$ 上。
1.  **载波功率 ($P_{c}$ 或 $P_{0T}$)**:
    $P_{c} = \frac{(V_{0}/\sqrt{2})^{2}}{R_{L}} = \frac{V_{0}^{2}}{2R_{L}}$
2.  **每个边频的功率 ($P_{USB}$, $P_{LSB}$)**:
    $P_{USB} = P_{LSB} = \frac{(m_{a}V_{0}/(2\sqrt{2}))^{2}}{R_{L}} = \frac{m_{a}^{2}V_{0}^{2}}{8R_{L}} = \frac{m_{a}^{2}}{4}P_{c}$
3.  **双边带总功率 ($P_{DSB}$)**:
    $P_{DSB} = P_{USB} + P_{LSB} = 2 \cdot \frac{m_{a}^{2}}{4}P_{c} = \frac{m_{a}^{2}}{2}P_{c}$
4.  **已调波总平均功率 ($P_{total}$ 或 $P_{o}$)**:
    $P_{total} = P_{c} + P_{USB} + P_{LSB} = P_{c} + \frac{m_{a}^{2}}{2}P_{c} = P_{c}(1 + \frac{m_{a}^{2}}{2})$

**功率效率 (Modulation Efficiency, $\eta$)**: 边带功率占总功率的百分比。
$\eta = \frac{P_{DSB}}{P_{total}} = \frac{m_{a}^{2}/2 \cdot P_{c}}{(1 + m_{a}^{2}/2)P_{c}} = \frac{m_{a}^{2}}{2+m_{a}^{2}}$
* 当 $m_{a}=1$ (100%调制) 时，$\eta = \frac{1}{2+1} = \frac{1}{3} \approx 33.3\%$。这意味着即使在最佳情况下，至少有 $2/3$ 的功率消耗在不携带信息的载波上。
* AM波的功率利用率较低，但其调制和解调电路简单，接收方便，因此广泛用于广播。

## 7.3 平方律调幅 (低电平调幅) 

平方律调幅是一种利用器件的非线性特性（特别是平方项）来实现振幅调制的方法，通常在低功率电平下进行，调制完成后再进行功率放大，因此称为低电平调幅。

### (1) 工作原理 
[Image 平方律调幅器原理框图 12]
将载波信号 $v_{c}(t) = V_{0}\cos(\omega_{0}t)$ 和调制信号 $v_{\Omega}(t) = V_{\Omega}\cos(\Omega t)$ 叠加后，加到一个非线性器件上。
输入信号 $v_{i}(t) = v_{c}(t) + v_{\Omega}(t)$。
非线性器件的输出 $v_{o}(t)$ 可以用泰勒级数展开：
$v_{o}(t) = a_{0} + a_{1}v_{i}(t) + a_{2}v_{i}^{2}(t) + a_{3}v_{i}^{3}(t) + \dots$

* **线性项 ($a_{1}v_{i}$)**: 产生 $\omega_{0}$ 和 $\Omega$ 频率分量。
* **平方项 ($a_{2}v_{i}^{2}$)**:
    $a_{2}v_{i}^{2} = a_{2}(V_{0}\cos(\omega_{0}t) + V_{\Omega}\cos(\Omega t))^{2}$
    $= a_{2}[V_{0}^{2}\cos^{2}(\omega_{0}t) + V_{\Omega}^{2}\cos^{2}(\Omega t) + 2V_{0}V_{\Omega}\cos(\omega_{0}t)\cos(\Omega t)]$
    $= a_{2}[\frac{V_{0}^{2}}{2}(1+\cos(2\omega_{0}t)) + \frac{V_{\Omega}^{2}}{2}(1+\cos(2\Omega t)) + V_{0}V_{\Omega}(\cos((\omega_{0}-\Omega)t) + \cos((\omega_{0}+\Omega)t))]$
    平方项产生了直流分量、二次谐波 $2\omega_{0}$ 和 $2\Omega$，以及重要的和差频 $(\omega_{0} \pm \Omega)$。这正是我们需要的边带分量。
* **三次项 ($a_{3}v_{i}^{3}$)**: 产生 $\omega_{0}$, $\Omega$, $3\omega_{0}$, $3\Omega$, $(\omega_{0} \pm 2\Omega)$, $(2\omega_{0} \pm \Omega)$ 等分量。其中 $(\omega_{0} \pm 2\Omega)$ 等分量靠近有用边带，难以用滤波器滤除，会造成失真。

为了获得纯净的AM波，理想的非线性器件应仅有平方项，即特性为 $v_{o} = a_{0} + a_{1}v_{i} + a_{2}v_{i}^{2}$。
输出信号经过一个中心频率为 $\omega_{0}$ 的带通滤波器，滤除不需要的频率分量（如直流、$\Omega$、$2\Omega$、$2\omega_{0}$），即可得到AM波：
$v_{AM}(t) = a_{1}V_{0}\cos(\omega_{0}t) + a_{2}V_{0}V_{\Omega}\cos((\omega_{0}-\Omega)t) + a_{2}V_{0}V_{\Omega}\cos((\omega_{0}+\Omega)t)$
这可以整理成标准AM波形式，其中载波幅值为 $a_{1}V_{0}$，调制度 $m_a = \frac{2a_{2}V_{\Omega}}{a_{1}}$。

平方律调幅器中的晶体管通常工作于甲类非线性状态，效率较低，适用于小信号调制。

### (2) 平衡调幅器 
平衡调幅器是一种改进的调制电路，可以用来抑制载波，从而产生**抑制载波双边带 (DSB-SC)**信号。
[Image 平衡调幅器电路（差分结构） 18]
假设两个特性相同的非线性器件 $D_1, D_2$，其电流特性为 $i = b_{0} + b_{1}v + b_{2}v^{2}$。
输入信号分别为：
$v_{1} = v_{c}(t) + v_{\Omega}(t) = V_{0}\cos(\omega_{0}t) + V_{\Omega}\cos(\Omega t)$
$v_{2} = v_{c}(t) - v_{\Omega}(t) = V_{0}\cos(\omega_{0}t) - V_{\Omega}\cos(\Omega t)$ (调制信号反相)
输出电压 $v_{o} = (i_{1}-i_{2})R$。
经过推导，输出电压 $v_{o}(t)$ 主要包含：
$v_{o}(t) = 2R[b_{1}V_{\Omega}\cos(\Omega t) + b_{2}V_{0}V_{\Omega}\cos((\omega_{0}+\Omega)t) + b_{2}V_{0}V_{\Omega}\cos((\omega_{0}-\Omega)t)]$
通过带通滤波器滤除低频的 $\Omega$ 项，即可得到：
$v_{DSB}(t) = 2Rb_{2}V_{0}V_{\Omega}[\cos((\omega_{0}-\Omega)t) + \cos((\omega_{0}+\Omega)t)]$
这个输出中不包含载波频率 $\omega_{0}$ 分量，只有上下边带，因此称为DSB-SC信号。平衡调幅器的优点是能有效地抑制载波。

## 双边带抑制载波调幅 (DSB-SC) 

### 1. 数学表达式 
DSB-SC信号可以通过调制信号与载波信号直接相乘得到：
$v_{DSB}(t) = k v_{\Omega}(t) \cdot v_{c}(t)$
其中 $k$ 是比例常数。
若调制信号为单音频 $v_{\Omega}(t) = V_{\Omega}\cos(\Omega t)$，载波为 $v_{c}(t) = V_{0}\cos(\omega_{0}t)$：
$v_{DSB}(t) = k V_{\Omega}V_{0}\cos(\Omega t)\cos(\omega_{0}t)$
$v_{DSB}(t) = \frac{1}{2}k V_{\Omega}V_{0}[\cos((\omega_{0}-\Omega)t) + \cos((\omega_{0}+\Omega)t)]$
可见，DSB-SC信号只包含上边带和下边带，没有载波分量。

### 2. 波形与频谱 
[Image DSB-SC信号的波形和频谱示意图 18] (Page 18 shows this for DSB)
* **波形**: DSB-SC信号的包络与调制信号的绝对值 $|v_{\Omega}(t)|$ 成正比。当调制信号过零时，已调波的相位会反转 $180^{\circ}$。因此，DSB-SC信号不仅是振幅调制，还包含了相位调制的信息。
* **频谱**: 只包含上边带 $(\omega_{0}+\Omega)$ 和下边带 $(\omega_{0}-\Omega)$。
* **功率**: 由于没有载波分量，DSB-SC的功率全部用于边带传输，功率利用率高于标准AM。总功率 $P_{DSB} = P_c \frac{k^2 V_{\Omega}^2}{2}$ (using a slightly different notation from source for clarity, assuming $V_0=1$ absorbed in $k$).
* **带宽**: 与AM相同，$BW = 2F_{max}$。

DSB-SC信号的解调通常需要同步检波（相干检波），即接收端需要一个与发射端载波同频同相的本地载波。

## 7.6 单边带信号 (SSB) 的产生 

SSB信号只传输一个边带（上边带或下边带），进一步提高了功率效率和频谱利用率。

### 1. 单边带通信的优缺点 
**优点**:
* **节省功率**: 相对于AM，理论上可节省至少 $5/6$ 的功率（载波占 $2/3$，一个边带占 $1/6$）。相对于DSB-SC，功率节省一半。
* **节省带宽**: 带宽减半，$BW = F_{max}$。这允许在相同频段内容纳更多信道。
* **提高信噪比**: 在相同峰值功率下，SSB接收端的信噪比可以提高。
* **减轻选择性衰落**: 由于只传输一个边带，由多径传播引起的边带间相位差异导致的衰落现象大大减轻。

**缺点**:
* 发送和接收设备复杂，对频率稳定度要求高，成本较高。SSB信号的解调也需要同步检波。

### 2. 产生SSB信号的方法 
主要有三种方法：滤波法、相移法和修正的相移滤波法（韦弗法）。

#### (1) 滤波法 (Filtering Method) 
[Image 滤波法产生SSB信号原理框图 21]
该方法首先产生一个DSB-SC信号，然后通过一个具有非常陡峭截止特性的边带滤波器，滤除其中一个边带，保留另一个边带。
* DSB信号: $v_{DSB}(t)=\frac{1}{2}kV_{\Omega}V_{0}[\cos((\omega_{0}+\Omega)t)+\cos((\omega_{0}-\Omega)t)]$
* 上边带信号 (USB): $v_{SSBU}(t)=\frac{1}{2}kV_{\Omega}V_{0}\cos((\omega_{0}+\Omega)t)$
* 下边带信号 (LSB): $v_{SSBL}(t)=\frac{1}{2}kV_{\Omega}V_{0}\cos((\omega_{0}-\Omega)t)$

[Image SSB信号的频谱（上边带和下边带） 21]

**对滤波器的要求**:
滤波器需要在载频 $\omega_0$ 附近将一个边带完整通过，同时完全抑制另一个非常靠近的边带。这要求滤波器具有极窄的过渡带。实现这种滤波器的难点在于过渡带与载频的相对值 $\Delta f / f_0$。当 $f_0$ 较高，或调制信号包含较低频率成分时（导致边带间隔 $2\Omega_{min}$ 很小），滤波器的设计和制造非常困难。通常，这种滤波器在较低的中频 (IF) 下更容易实现。
因此，实际中可能需要多次混频和滤波：先在较低频率产生SSB信号，然后通过多次上变频将其转换到最终的发射频率。
[Image 多级平衡调幅和滤波产生高频SSB信号框图 23]

#### (2) 相移法 (Phase-Shift Method) 
[Image 相移法产生SSB信号原理框图 24]
相移法利用信号的希尔伯特变换特性，通过移相来抵消不需要的边带。
调制信号 $v_{\Omega}(t)$ 和其 $90^{\circ}$ 相移版本 $v'_{\Omega}(t)$ (希尔伯特变换)；载波 $v_c(t)$ 和其 $90^{\circ}$ 相移版本 $v'_c(t)$。
例如，若 $v_{\Omega}(t) = V_{\Omega}\cos(\Omega t)$ 和 $v_c(t) = V_0\cos(\omega_0 t)$：
$v'_{\Omega}(t) = V_{\Omega}\sin(\Omega t)$ (实际应为 $\cos(\Omega t - 90^\circ)$, 但用 $\sin$ 更常见于表达式)
$v'_c(t) = V_0\sin(\omega_0 t)$

一个平衡调幅器输入 $v_{\Omega}(t)$ 和 $v_c(t)$，输出 $v_1(t) = k \cos(\Omega t)\cos(\omega_0 t)$。
另一个平衡调幅器输入 $v'_{\Omega}(t)$ 和 $v'_c(t)$，输出 $v_2(t) = k \sin(\Omega t)\sin(\omega_0 t)$。
(注意图 中是 $v_1=V~sin~\Omega tsin~\omega_{0}t$ 和 $v_2=V~cos~\Omega tcos~\omega_{0}t$)

根据三角恒等式:
$\cos A \cos B \mp \sin A \sin B = \cos(A \pm B)$
$v_{SSB}(t) = v_1(t) \pm v_2(t)$
若取 $v_1(t) = V\cos(\Omega t)\cos(\omega_0 t)$ 和 $v_2(t) = V\sin(\Omega t)\sin(\omega_0 t)$：
* $v_{SSBU}(t) = V\cos(\Omega t)\cos(\omega_0 t) - V\sin(\Omega t)\sin(\omega_0 t) = V\cos((\omega_0+\Omega)t)$ (上边带)
* $v_{SSBL}(t) = V\cos(\Omega t)\cos(\omega_0 t) + V\sin(\Omega t)\sin(\omega_0 t) = V\cos((\omega_0-\Omega)t)$ (下边带)

[Image 利用加减法器和乘法器的相移法SSB产生电路 25]
相移法的关键在于实现宽带 $90^{\circ}$ 相移网络，尤其对于调制信号（通常是语音这类宽带信号）而言比较困难。

#### (3) 修正的相移法 (Modified Phase-Shift Method / Weaver's Method) 
[Image 修正的相移法（韦弗法）SSB产生框图 26]
这种方法也称为第三种方法或韦弗法，它结合了滤波和相移的思想，可以避免对音频信号进行宽带 $90^{\circ}$ 相移的困难。它使用两个平衡调幅器、两个低通滤波器和两个正交的载波源。

## 7.8 高电平调幅 

振幅调制电路的核心是实现调制信号与载波信号的乘积项。
**调制分类**:
* **高电平调制**: 调制过程发生在末级功率放大器中，调制器本身需要提供较大的功率。主要用于产生AM信号，如无线电广播。
* **低电平调制**: 先完成调制，再进行多级放大以达到所需的发射功率。主要用于DSB、SSB以及FM信号。

高电平AM调制通常在C类功率放大器中进行。主要有集电极调幅和基极调幅。

### 1. 集电极调幅 (Collector AM)
[Image 集电极（或漏极）调幅电路图 29, 32]
在集电极调幅电路中，调制信号 $v_{\Omega}(t)$ 叠加到功率放大管的集电极直流电源 $V_{CC}$ 上，使得集电极的有效电源电压随调制信号变化。
有效集电极电源 $E_{C}(t) = V_{CC} + v_{\Omega}(t) = V_{CC} + V_{\Omega m}\cos(\Omega t) = V_{CC}(1 + m_{a}\cos(\Omega t))$，其中调制度 $m_{a} = V_{\Omega m}/V_{CC}$。
功率放大管（通常工作在丙类或乙类）的输出电压幅值将随 $E_{C}(t)$ 变化，从而实现振幅调制。

**能量关系** (以丙类放大器为例):
* 集电极电流直流分量 $I_{co}(t) = I_{coT}(1+m_{a}\cos(\Omega t))$ ( $I_{coT}$ 为未调时直流分量)。
* 集电极电流基波分量 $I_{cm1}(t) = I_{cm1T}(1+m_{a}\cos(\Omega t))$ ( $I_{cm1T}$ 为未调时基波幅值)。
* $E_C$ 提供的总平均功率: $P_{=av} = \frac{1}{2\pi}\int_{-\pi}^{\pi}E_{C}(t)I_{CO}(t)d(\Omega t) = P_{=T}(1+\frac{m_{a}^{2}}{2})$，其中 $P_{=T}=V_{CC}I_{coT}$ 是未调时直流电源功率。
* 调制信号源提供的平均功率: $P_{c\Omega} = P_{=av} - P_{=T} = \frac{1}{2}m_{a}^{2}P_{=T}$。
* 输出的平均射频功率: $P_{oav} = P_{oT}(1+\frac{1}{2}m_{a}^{2})$，其中 $P_{oT} = \frac{1}{2}I_{cm1T}^{2}R_{p}$ 是未调时输出载波功率 ($R_p$ 为等效负载电阻)。
* 平均效率: $\eta_{av} = \frac{P_{oav}}{P_{=av}} = \frac{P_{oT}(1+\frac{1}{2}m_{a}^{2})}{P_{=T}(1+\frac{m_{a}^{2}}{2})} = \frac{P_{oT}}{P_{=T}} = \eta_{T}$ (未调时效率)。
    集电极调幅的效率与调制度 $m_a$ 无关，等于未调制时的效率。这使得晶体管能得到充分利用，可获得较高功率输出。

**晶体管耐压**:
集电极-发射极间最大瞬时电压 $v_{cEmax} = (V_{CC}+V_{cT})(1+m_{a})$，其中 $V_{cT}$ 是输出回路中载波电压幅值。
若放大器工作在临界状态或轻微欠压状态，则 $V_{cT} \approx V_{CC}$。
当 $m_{a}=1$ 时，$v_{cEmax} \approx (V_{CC}+V_{CC})(1+1) = 4V_{CC}$。
选管时，其耐压必须大于 $4V_{CC}$，功耗则按 $P_{cav}$ (平均集电极耗散功率) 选取。

**特点**:
* 效率高，可获得较大输出功率。
* 调制信号源需要提供较大的功率 ($P_{c\Omega}$)。
* 当 $E_c$ 较小时（对应调制信号的谷值），放大器可能进入严重过压区，导致 $I_{cm1}$ 急剧下降，产生非线性失真；当 $E_c$ 较大时， $E_c$ 对 $I_{cm1}$ 的控制可能减弱。

### 2. 基极调幅 (Base AM)
[Image 基极（或栅极）调幅电路图 34, 37]
在基极调幅电路中，调制信号 $v_{\Omega}(t)$ 叠加到功率放大管的基极偏置电压 $-V_{BB}$ 上，使得有效基极偏置 $V_{B}(t) = -V_{BB} + v_{\Omega}(t)$ 随调制信号变化。
基极偏置的变化会改变放大管的导通角和集电极电流脉冲的幅值，从而改变输出谐振回路中高频电压的幅值，实现振幅调制。

**工作状态选择**:
基极调幅电路通常使放大器工作在**欠压状态**。在欠压区，集电极电流基波幅值 $I_{cm1}$ 近似与基极有效偏压 $V_B$ 成线性关系 ($I_{cm1} \propto V_B$)。
当调制信号使 $V_B$ 达到最大值 $V_{Bmax}$ 时，放大器的工作状态不应超过临界状态，以避免饱和或过驱动引起的严重失真。
[Image 基极调幅时集电极电流基波幅值 $I_{cm1}$ 随基极偏压 $V_B$ 变化的特性曲线 35]

**特点** :
* 所需调制信号功率较小，因为调制信号加在输入回路。
* 与集电极调幅相比，工作效率较低（因为通常在欠压区，且偏压变化范围较大）。
* 输出波形失真可能较大，线性度不如集电极调幅。

## 7.9 包络检波 (Envelope Detection)

解调是调制的逆过程，是从高频已调波中恢复出原低频调制信号的过程。从频谱上看，解调也是一种信号频谱的线性搬移过程，是将高频端的信号频谱搬移到低频端。不同的调制方式对应于不同的解调方法。 [cite: 2]

振幅调制信号主要有 AM (普通调幅)、DSB (双边带抑制载波) 和 SSB (单边带)。
解调方法主要有：
* **包络检波**：适用于AM信号。 [cite: 2]
* **同步检波**：适用于AM、DSB、SSB信号。

**包络检波器基本原理**：
包络检波器通过非线性电路（如二极管）和低通滤波器来提取已调AM信号的包络。 [cite: 2]
[Image Envelope detection waveform 2]
其基本构成如下图所示：
输入AM信号 $\rightarrow$ 非线性电路 $\rightarrow$ 低通滤波器 $\rightarrow$ 检出包络信息 [cite: 2]

### 二极管峰值包络检波器

这是一种常见的包络检波电路。

**1. 基本电路结构** [cite: 3]
[Image Diode peak envelope detector circuit 3]
* **输入回路**：通常是中频放大器的输出回路，提供AM信号 $v_i$。
* **二极管D**：作为非线性元件，实现整流功能。
* **输出回路**：由电阻R和电容C组成的低通RC滤波网络。

**RC网络的基本功能**： [cite: 3]
* 对低频调制信号 $v_{\Omega}$ (角频率 $\Omega$)，RC网络应呈现较大的阻抗，通常要求 $\frac{1}{\Omega C} \gg R$，此时电容C近似开路，检波负载主要为R。
* 对高频载波信号 (角频率 $\omega_0$)，RC网络应呈现很小的阻抗，将其滤除，通常要求 $\frac{1}{\omega_0 C} \ll R$。
* 理想情况下，RC网络的阻抗为：
    $Z_{RC}(\omega) = \begin{cases} Z(\omega_0) \approx 0 & \text{(对载波短路)} \\ Z(\Omega) = R & \text{(对调制信号为R)} \end{cases}$ [cite: 3]

**峰值检波器的工作条件**： [cite: 3]
检波器一般工作在大信号状态，输入信号幅值要求大于二极管的开启电压（例如对于硅管大于0.5V-0.7V），通常在1V左右。

**2. 工作原理分析** [cite: 4]
[Image Peak detector waveforms 4]
* **(1) 充电过程**：在输入AM信号 $v_i$ 的正半周，如果 $v_i$ 大于输出电压 $v_o$ 加上二极管的导通压降，二极管D正向偏置而导通。电流 $i_D$ 对电容C充电。由于充电时间常数 $r_d C$ (其中 $r_d$ 是二极管导通电阻) 很小，充电迅速，使输出电压 $v_o$ 很快接近 $v_i$ 的峰值，即 $v_o \approx v_i$ (忽略导通压降时)。 [cite: 4] 二极管两端的电压为 $v_D = v_i - v_o$。D的导通与截止取决于 $v_D$ 的极性。 [cite: 4]
* **(2) 放电过程**：当 $v_i$ 从峰值下降，使得 $v_i < v_o$ 时 (或 $v_i - v_o < V_{on}$)，二极管D截止。电容C通过负载电阻R放电。放电时间常数 $\tau = RC$ 通常选择得远大于载波周期 $T_0$ ($RC \gg T_0$，其中 $T_0 = \frac{1}{f_0} = \frac{2\pi}{\omega_0}$)，因此放电较缓慢，使得 $v_o$ 能在载波的间歇期间基本保持 $v_i$ 的峰值，但略有下降。 [cite: 4]
* **(3) 再充电**：当 $v_i$ 的下一个正半周到来，且 $v_i > v_o$ 时，D又导通，再次对C充电，使 $v_o$ 跟踪 $v_i$ 新的峰值。 [cite: 4]
如此循环，输出电压 $v_o$ 的波形便能基本跟随输入AM信号的包络。输出电压 $v_o$ 包含直流分量 $V_{DC}$ 和恢复的低频调制信号 $v_{\Omega}$，即 $v_o = V_{DC} + v_{\Omega}$。 [cite: 4]

**3. 应用输出电路** [cite: 5]
[Image Output circuits (a) and (b) 5]
* 图(a) 电路：通过隔直电容 $C_g$ 输出恢复的调制信号 $v_{\Omega}$，作为检波电路的典型输出。 [cite: 5]
* 图(b) 电路：输出与已调波包络（进而与载波大小）成正比的直流分量 $V_{DC}$，可作为自动增益控制(AGC)信号。 [cite: 5]

**4. 电路性能分析**

* **(1) 传输系数 (检波效率 $K_d$)** [cite: 6, 7, 8]
    定义：$K_d = \frac{\text{输出低频交流电压振幅}}{\text{输入已调波包络振幅}} = \frac{V_{\Omega}}{mV_c}$ (其中 $V_c$ 是输入未调载波的振幅，$m$ 是调幅指数)。 [cite: 6]
    设输入AM信号为 $v_i = V_c(1+m \cos\Omega t)\cos\omega_0 t$。 [cite: 6, 7]
    输出信号近似为 $v_o(t) \approx V_c(1+m \cos\Omega t)\cos\theta$，其中 $\theta$ 是二极管的导通角。 [cite: 7]
    则 $v_o(t)$ 包含直流分量 $V_{DC} = V_c \cos\theta$ 和低频调制分量 $v_{\Omega}(t) = mV_c \cos\theta \cos\Omega t$。 [cite: 7]
    因此，低频交流电压振幅 $V_{\Omega} = mV_c \cos\theta$。 [cite: 7]
    所以检波效率 $K_d = \frac{mV_c \cos\theta}{mV_c} = \cos\theta$。 [cite: 8]
    [Image Diode characteristic and waveforms 8]
    导通角 $\theta$ 的近似表达式为 $\theta \approx \sqrt[3]{\frac{3\pi R_D}{R}}$，其中 $R_D$ 是二极管导通时的平均内阻，$R$ 是负载电阻。 [cite: 8]
    * **讨论**： [cite: 9]
        * 在大信号检波时，当D和R确定后，$\theta$ 近似为恒定值，与输入信号大小关系不大，因此 $K_d = \cos\theta$ 也近似恒定。输入包络与输出低频信号之间呈线性关系，故称为线性检波。 [cite: 9] 此时输出信号为 $v_o(t) = K_d V_c(1+m \cos\Omega t)$ (指包络部分)。 [cite: 9]
        * 当 $R \uparrow \implies \theta \downarrow \implies K_d = \cos\theta \uparrow$。但 $K_d$ 总是小于1。理想情况下 $K_d=1$。一般当 $R/R_D > 50$ (即 $gR > 50$) 时，$K_d > 0.9$。 [cite: 9]

* **(2) 检波器的等效输入电阻 $R_{id}$** [cite: 9, 10]
    峰值检波器作为前级（如中放末级）的负载，其输入阻抗会影响前级的有载Q值及回路特性。 [cite: 9]
    [Image Detector as load to IF stage 10]
    $R_{id}$ 是指检波器对载波频率信号呈现的等效电阻。若输入为等幅载波信号 $v_i = V_{im}\cos\omega_0 t$。 [cite: 9]
    忽略二极管导通电阻上的功耗，由能量守恒，输入到检波器的高频功率应等于输出端负载电阻 $R_L$ (即检波电路中的R) 上消耗的直流功率。 [cite: 10]
    $\frac{V_{im}^2}{2R_{id}} = \frac{V_o^2}{R_L}$ (注意 $V_o$ 是输出的直流电压)。 [cite: 10]
    由于 $V_o \approx K_d V_{im}$，在大信号且 $K_d \approx 1$ 时，$V_o \approx V_{im}$。
    则 $R_{id} \approx \frac{1}{2}R_L$。 [cite: 10] 这表明检波器从输入端看等效为一个约为负载电阻一半的电阻。

* **(3) 失真**
    * **① 频率失真 (线性失真)** [cite: 11]
        由电路中的线性元件（R, C, $C_c$）的频率特性不理想引起，导致对不同频率的调制信号分量有不同的衰减或相移。
        [Image Circuit for frequency distortion analysis 11]
        * **滤波电容C**：影响高频调制分量 $\Omega_{max}$。为保证 $\Omega_{max}$ 不被C过度旁路，要求C不能太大，即 $\frac{1}{\Omega_{max}C} \gg R$ (通常10倍以上)，或 $C \ll \frac{1}{\Omega_{max}R}$。 [cite: 11] (更准确的条件是RC时间常数与最高调制频率的关系，见设计原则)
        * **耦合电容 $C_c$**：影响低频调制分量 $\Omega_{min}$。为保证 $\Omega_{min}$ 在 $C_c$ 上的压降不大，要求 $C_c$ 足够大，即 $\frac{1}{\Omega_{min}C_c} \ll r_{i2}$ (其中 $r_{i2}$ 是下一级的输入电阻)。 [cite: 12]

    * **② 非线性失真** [cite: 12]
        当输入信号幅值 $V_{im}$ 较小或负载电阻R较小时，非线性失真会增大。 [cite: 12]
        * 原因(a)：$V_{im}$ 较小时，二极管工作在其伏安特性的弯曲部分（小信号检波），导致输出与输入包络不成线性关系。 [cite: 12]
        * 原因(b)：R较小时，输出电压 $v_o$ 较小，对二极管的偏置作用减弱，等效于工作点变化，也可能引入非线性。 [cite: 12]
        * 为减小非线性失真，应使输入信号幅值足够大 ($V_{im} \uparrow$)，且负载电阻R取值适当 ($R \uparrow$)。 [cite: 12]

    * **③ 惰性失真 (Diagonal Clipping 或 斜率过载失真)** [cite: 13]
        [Image Inertial distortion waveform 14]
        当RC取值过大，使得RC放电时间常数过长，输出电压 $v_c$ (电容C上的电压) 的下降速度跟不上输入AM信号包络的下降速度时，就会产生惰性失真。此时二极管在包络下降期间长时间截止，输出波形不能正确反映包络变化。 [cite: 13]
        产生原因：输入AM信号包络的变化率 $> RC$ 放电的速率。 [cite: 13]
        避免条件：在任何时刻，电容C上电压的放电变化率（绝对值）应大于或等于包络信号的变化率（绝对值）。 [cite: 14]
        $|\frac{dv_c}{dt}| \ge |\frac{dv_{\text{env}}(t)}{dt}|$
        其中 $v_{\text{env}}(t) = V_c(1+m_a \cos\Omega t)$。
        $|\frac{dv_c}{dt}| \approx \frac{v_c}{RC} \approx \frac{V_c(1+m_a \cos\Omega t)}{RC}$ (放电时)。 [cite: 15]
        $|\frac{dv_{\text{env}}(t)}{dt}| = |-m_a V_c \Omega \sin\Omega t| = m_a V_c \Omega |\sin\Omega t|$。 [cite: 14]
        不产生惰性失真的条件为： $RC\Omega_{max} \le \frac{\sqrt{1-m_a^2}}{m_a}$ (其中 $\Omega_{max}$ 是最高调制角频率，$m_a$ 是调幅指数)。  [cite: 16]

    * **④ 负峰切割失真 (Bottom Clipping)** [cite: 16]
        [Image Circuit for negative peak clipping 16]
        [Image Negative peak clipping waveform 17]
        当通过耦合电容 $C_c$ 连接下一级时，检波器的直流负载电阻为 $R$，而交流负载电阻为 $R_{\Omega} = \frac{R \cdot r_{i2}}{R+r_{i2}}$ (其中 $r_{i2}$ 是下一级输入电阻)。 [cite: 16] 由于 $R_{\Omega} < R$，导致交流负载线和直流负载线不同。
        在信号包络的负峰处，如果调制深度 $m_a$ 较大，可能导致二极管提前截止，使得输出信号的负半周被切割。
        不产生负峰切割失真的条件为：$m_a \le \frac{R_{\Omega}}{R}$ 或 $m_a \le \frac{r_{i2}}{R+r_{i2}}$。 [cite: 17]
        **改善方法**： [cite: 18]
        * 在检波器与下级放大器之间加射极跟随器以提高 $r_{i2}$。 [cite: 18]
        * 将检波负载R分成两部分 $R_1$ 和 $R_2$ (如P18图)，使直流负载为 $R_1+R_2$，交流负载为 $R_1 + \frac{R_2 r_{i2}}{R_2+r_{i2}}$，从而调整交直流负载比。 [cite: 18]

**5. 检波器设计及元件参数选择的原则** [cite: 19]
[Image Detector circuit with IF stage parameters 19]
* **(1) 输入回路有载Q值 $Q_L$**：应足够大以满足选择性和通频带要求。$Q_L = \omega_0 C_s (R_s || \frac{R_{load}}{2}) \gg 1$ (其中 $R_{load}$ 为检波器负载电阻R，$\frac{R_{load}}{2}$ 为检波器输入电阻)。 [cite: 19]
* **(2) 高频纹波**：为保证输出的高频纹波小，要求 $RC \gg T_0$，即 $RC \gg \frac{1}{\omega_0}$ (或 $\frac{2\pi}{\omega_0}$)。 [cite: 19]
* **(3) 低频频率失真**：
    * 为了不影响调制信号的上限频率 $\Omega_{max}$ (高频特性好)：$RC \le \frac{1}{\Omega_{max}}$ (即RC时间常数要小于最高调制信号周期的约1/ $(2\pi)$，确保电容电压能跟随最快包络变化)。 [cite: 19]
    * 为了不影响调制信号的下限频率 $\Omega_{min}$ (低频特性好，指耦合电容 $C_g$ 和下级输入电阻 $R_g=r_{i2}$ )：$R_g C_g \gg \frac{1}{\Omega_{min}}$ (即耦合时间常数远大于最低调制信号周期的约1/ $(2\pi)$)。 [cite: 19]
* **(4) 惰性失真**：为避免惰性失真，要求 $RC \Omega_{max} \le \frac{\sqrt{1-m_a^2}}{m_a}$ (这里的 $m_a$ 是最大调幅指数 $m_{max}$)。 [cite: 19]
* **(5) 底部切割失真**：为避免底部切割失真，要求 $m_a \le \frac{R_g}{R+R_g}$ (其中 $R_g = r_{i2}$)。 [cite: 19]

这些条件之间往往存在一定的制约，需要综合考虑进行优化选择。

## 7.10 同步检波 (Synchronous Detection)

对于DSB (双边带抑制载波) 和SSB (单边带)信号，其包络不再是原始调制信号的形状，因此不能用简单的包络检波器进行解调。必须使用同步检波器。 [cite: 23]
同步检波的关键在于接收端需要提供一个与原始调制载波**同步**（即同频同相）的本地载波信号。 [cite: 23]
[Image AM and DSB demodulation block diagrams 23]

同步检波器可分为：**乘积型同步检波器**和**叠加型同步检波器**。 [cite: 24]

### 1. 乘积型同步检波器 [cite: 24]
[Image Product type synchronous detector block diagram 24]
其核心是将输入的已调信号与本地产生的同步载波相乘，然后通过低通滤波器(LPF)提取出原始调制信号。

设输入已调波为DSB信号：$v_{DSB}(t) = V_m(t) \cos\omega_0 t = V \cos\Omega t \cos\omega_0 t$ (单音频调制时)。 [cite: 24]
本地恢复的载波为：$v_{lo}(t) = V_{LO} \cos(\omega_1 t + \varphi)$。 [cite: 24]
相乘器输出为：
$v_p(t) = v_{DSB}(t) \cdot v_{lo}(t) = V \cos\Omega t \cos\omega_0 t \cdot V_{LO} \cos(\omega_1 t + \varphi)$
$v_p(t) = \frac{1}{2} V V_{LO} \cos\Omega t [\cos((\omega_0 - \omega_1)t - \varphi) + \cos((\omega_0 + \omega_1)t + \varphi)]$ [cite: 24]

令 $\Delta\omega = \omega_0 - \omega_1$ (频率差)，则经低通滤波器滤除高频分量 $(\omega_0 + \omega_1)$ 后，输出信号为：
$v_{\Omega}(t) = \frac{1}{4} V V_{LO} \cos\Omega t \cos(\Delta\omega t - \varphi)$ (注意：课件中是 $\frac{1}{2} V V_0 \cos\Omega t \cos(\Delta\omega_0 t + \varphi)$，这里的系数是基于课件公式的直接结果，但标准的DSB解调输出通常是 $\frac{1}{2} k V V_{LO} \cos\Omega t \cos(\Delta\omega t - \varphi)$，k是乘法器和LPF的增益。课件中的公式推导展开 $\cos\omega_0 t \cos(\omega_1 t + \varphi)$ 已经包含了一个 $\frac{1}{2}$。)
课件结果为：$v_{\Omega}(t) = \frac{1}{2} V V_{LO} \cos\Omega t \cos(\Delta\omega t - \varphi)$ (沿用课件中的形式，并假设其 $V_0$ 为 $V_{LO}$)。 [cite: 24]

**讨论**： [cite: 25]
* **(1) 完全同步**：当本地载波与调制载波同频同相，即 $\Delta\omega = 0$ 且 $\varphi = 0$。
    $v_{\Omega}(t) = \frac{1}{2} V V_{LO} \cos\Omega t$。 [cite: 25]
    此时，解调输出信号与原始调制信号 $\cos\Omega t$ 成正比，实现了无失真解调。
* **(2) 频率误差**：当 $\Delta\omega \ne 0$ (即使 $\varphi = 0$)。
    $v_{\Omega}(t) = \frac{1}{2} V V_{LO} \cos\Omega t \cos(\Delta\omega t)$。 [cite: 25]
    输出信号的幅度会随着 $\cos(\Delta\omega t)$ 变化，产生一种称为“拍频”的失真。如果 $\Delta\omega$ 较大，信号会被严重破坏。
* **(3) 相位误差**：当 $\varphi \ne 0$ (即使 $\Delta\omega = 0$)。
    $v_{\Omega}(t) = \frac{1}{2} V V_{LO} \cos\varphi \cos\Omega t$。 [cite: 25]
    输出信号的幅度会乘以一个衰减因子 $\cos\varphi$。如果相位差 $\varphi = \frac{\pi}{2}$（正交），则 $\cos\varphi = 0$，输出信号为零，称为“正交零点效应”。如果相位 $\varphi$ 随时间缓慢变化，则输出幅度也会缓慢变化。

[Image DSB spectrum and LPF output 25]

**注意点**： [cite: 26]
* 同步解调的关键是实现精确的乘法运算和有效的低通滤波。 [cite: 26]
* 同步检波能否无失真恢复信号的关键在于本地载波与原始载波的同步精度。实际中常采用锁相环(PLL)或Costas环来提取同步载波。 [cite: 26]

### 2. 叠加型同步检波器 [cite: 26]
这种方法通常用于SSB信号的解调，其原理是在接收端将SSB信号与一个幅度远大于它的本地同步载波相加，然后通过包络检波器来恢复原信号。
[Image Superposition type synchronous detector block diagram 26]

设输入为上边带SSB信号：$v_1(t) = V_1 \cos((\omega_0 + \Omega)t)$。 [cite: 27]
本地同步载波：$v_{lo}(t) = V_{LO} \cos\omega_0 t$ (课件用 $V_o$ 表示 $V_{LO}$)。 [cite: 27]
两者相加得到：$v_2(t) = v_1(t) + v_{lo}(t) = V_1 \cos((\omega_0 + \Omega)t) + V_{LO} \cos\omega_0 t$
$v_2(t) = V_1(\cos\omega_0 t \cos\Omega t - \sin\omega_0 t \sin\Omega t) + V_{LO} \cos\omega_0 t$
$v_2(t) = (V_{LO} + V_1 \cos\Omega t)\cos\omega_0 t - (V_1 \sin\Omega t)\sin\omega_0 t$ [cite: 27]

这是一个包络和相位都受到 $\Omega t$ 调制的信号。其包络 $V_m(t)$ 为：
$V_m(t) = \sqrt{(V_{LO} + V_1 \cos\Omega t)^2 + (V_1 \sin\Omega t)^2}$ [cite: 28]
$V_m(t) = \sqrt{V_{LO}^2 + 2V_{LO}V_1 \cos\Omega t + V_1^2 \cos^2\Omega t + V_1^2 \sin^2\Omega t}$
$V_m(t) = \sqrt{V_{LO}^2 + V_1^2 + 2V_{LO}V_1 \cos\Omega t}$
$V_m(t) = V_{LO} \sqrt{1 + (\frac{V_1}{V_{LO}})^2 + 2\frac{V_1}{V_{LO}} \cos\Omega t}$ [cite: 28]

当本地载波幅度远大于信号幅度，即 $V_{LO} \gg V_1$ 时：
$(\frac{V_1}{V_{LO}})^2$ 项可以忽略，或者使用二项式展开 $\sqrt{1+x} \approx 1 + \frac{1}{2}x - \frac{1}{8}x^2 + \dots$
$V_m(t) \approx V_{LO} (1 + \frac{V_1}{V_{LO}} \cos\Omega t) = V_{LO} + V_1 \cos\Omega t$。 [cite: 28] (这是最简化的近似)
课件中给出的更详细的展开，保留了 $(\frac{V_1}{V_{LO}})^2$ 的项：
$V_m(t) \approx V_{LO} [1 - \frac{1}{4}(\frac{V_1}{V_{LO}})^2 + \frac{V_1}{V_{LO}}\cos\Omega t - \frac{1}{4}(\frac{V_1}{V_{LO}})^2 \cos2\Omega t]$ [cite: 28]
通过包络检波器检出 $V_m(t)$ 后，其直流分量 $V_{LO}(1 - \frac{1}{4}(\frac{V_1}{V_{LO}})^2)$ 可以被隔掉，期望的信号项是 $V_1 \cos\Omega t$ (课件中是 $V_{LO} \cdot \frac{V_1}{V_{LO}}\cos\Omega t$)。同时存在一个二阶失真项 $\cos2\Omega t$。 [cite: 29]
为了减小失真，要求 $V_{LO} \gg V_1$，使得 $k_{f2} = \frac{1}{4}\frac{V_1}{V_{LO}} \ll 1$。 [cite: 29]

**要求**： [cite: 29]
* 本地载波的频率和相位必须与发射端的原始载波严格同步。
* 一种实现方法是在发射端发送一个小的导频(pilot tone)，接收端用窄带滤波器将其取出并放大，作为本地同步载波。 [cite: 29]

### 补充：典型无线电系统框图

**调幅广播发射机方框图** [cite: 30]
[Image AM Broadcast Transmitter Block Diagram 30]
基本组成：
声音信号 $\rightarrow$ 话筒 $\rightarrow$ 音频放大器 $\rightarrow$ 调制器 $\rightarrow$ 高频功率放大器 $\rightarrow$ 天线
同时，高频振荡器产生载波信号，送至调制器（低电平调制时）或高频功率放大器（高电平调制时）。

**超外差收音机方框图** [cite: 31]
[Image Superheterodyne Receiver Block Diagram 31]
基本组成：
天线 $\rightarrow$ 高频放大器 (RF Amp) $\rightarrow$ 混频器 (Mixer) $\rightarrow$ 中频放大器 (IF Amp) $\rightarrow$ 检波器 (Detector) $\rightarrow$ 低频放大器 (AF Amp) $\rightarrow$ 扬声器
本地振荡器 (Local Oscillator) 为混频器提供本机振荡信号。
核心部分是**混频器**，它将接收到的不同频率的射频信号变换为一个固定的中频信号（外差作用），便于后续的固定频率放大和滤波，从而提高收音机的灵敏度和邻道选择性。 [cite: 31]
