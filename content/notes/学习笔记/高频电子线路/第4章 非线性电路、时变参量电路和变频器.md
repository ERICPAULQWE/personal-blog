## 4.1 概述 

**频率变换**：指输出信号的频率与输入信号的频率不同，而且满足一定的变换关系。 

频率变换电路分为： 

* **频谱搬移电路（线性频率变换）** 
    * **作用**：将输入信号频谱沿频率轴进行不失真的搬移，搬移前后各频率分量的相对大小和频谱内部结构（相互间隔）保持不变。这广泛应用于超外差式接收机中的变频、调幅信号的检波等场合。 
* **频谱非线性变换电路（非线性频率变换）** 
    * **作用**：将输入信号频谱进行特定的非线性变换，例如产生新的频率分量，如谐波或组合频率。这常用于调频、鉴频等电路中。 

## 4.2 非线性电路的分析方法 

**线性元件**：元件的参数（如电阻值 $R$、电容值 $C$、电感值 $L$）与通过元件的电流 $i$ 或加在元件两端的电压 $v$ 无关，其伏安特性曲线是直线。例如理想的电阻、电容、电感。 
* **特点**：元件参数为常数，电路分析时满足叠加定理，本身不能实现频率变换作用。 

**非线性元件**：元件的参数与通过元件的电流 $i$ 或加在元件两端的电压 $v$ 有关，其伏安特性曲线不是直线。例如二极管、晶体管、带磁心的电感等。 
* **特点**：具有工作特性的非线性；不满足叠加定理；能够实现频率变换作用。这是因为非线性元件可以将输入信号的不同频率分量进行混合，从而产生新的频率分量。 

**线性电路**：由线性元件组成的电路。 
**非线性电路**：含有一个或多个非线性元件的电路。 

### 非线性元件的频率变换作用 
当非线性元件的伏安特性可以用幂级数表示时，例如 $i = kv^2$ 
或更一般的形式 $i = k_0 + k_1v + k_2v^2 + k_3v^3 + \dots$ ，
如果输入电压 $v$ 是由两个不同频率的正弦信号 $v_1 = V_{1m}(1+\sin\omega_1t)$ 和 $v_2 = V_{2m}(1+\sin\omega_2t)$ 叠加而成，
即 $v_{\Sigma} = v_1 + v_2$。 

那么输出电流 $i_{\Sigma}$ 将包含原始频率、它们的谐波以及它们的和频与差频等新的频率分量。
例如，对于 $i=kv^2$ ：
$v_{\Sigma} = V_{1m} + V_{2m} + V_{1m}\sin\omega_1t + V_{2m}\sin\omega_2t$ 
$i_{\Sigma} = kv_{\Sigma}^2 = k[(V_{1m}+V_{2m})^2 + V_{1m}^2\sin^2\omega_1t + V_{2m}^2\sin^2\omega_2t + 2(V_{1m}+V_{2m})(V_{1m}\sin\omega_1t + V_{2m}\sin\omega_2t) + 2V_{1m}V_{2m}\sin\omega_1t\sin\omega_2t + \dots]$ 
其中，利用三角函数积化和差公式：
$\sin\omega_1t \sin\omega_2t = \frac{1}{2}[\cos(\omega_1-\omega_2)t - \cos(\omega_1+\omega_2)t]$ （注意：原始文档中为 $\frac{1}{2}[\cos(\omega_1+\omega_2)t + \cos(\omega_1-\omega_2)t]$，这取决于具体展开，但通常取差频和和频）。 
这表明输出电流中包含了 $(\omega_1+\omega_2)$ 和 $(\omega_1-\omega_2)$ 的频率分量，实现了频率变换。

对于 $i=k_0+k_1v+k_2v^2+k_3v^3$ ：
$i_3 = kv^3 = k[(v_1+v_2)^3] = k[v_1^3 + 3v_1^2v_2 + 3v_1v_2^2 + v_2^3]$ 
这类三次项会产生更复杂的组合频率，如 $2\omega_1 \pm \omega_2$ 和 $\omega_1 \pm 2\omega_2$ 等。

### 非线性电路的分析方法 
主要有以下几种方法：
* **图解法**：根据非线性元件的特性曲线和输入信号波形，通过作图直接求出电路中的电流和电压波形。这种方法直观，但精度有限，适用于一些具有明显非线性特性的元件，如电子管。 
* **解析法**：将非线性元件的特性曲线近似地表示为数学解析式，列出电路方程，从而解得电路中的电流和电压。这种方法精度较高，但求解过程可能复杂。 
    * **幂级数法**：将非线性特性在工作点附近展开成泰勒级数（幂级数）。适用于输入信号幅度较小的情况，如混频、小信号检波。 
        数学表达式为： $i = b_0 + b_1(v-V_0) + b_2(v-V_0)^2 + b_3(v-V_0)^3 + \dots$ 
        当信号幅度较小时，只需取幂级数的前几项（通常到二次项或三次项）。 
    * **折线法**：将非线性特性曲线用若干段直线来近似。适用于输入信号幅度较大的情况，如高频功率放大、大信号检波。 
        例如，晶体管工作在开关状态或接近开关状态时，其导通角 $\theta_c$ 可以计算：
        $\theta_c = \cos^{-1}\left[\frac{V_{BZ}+V_{BB}}{V_{bm}}\right]$ 
        在导通角内，集电极电流 $i_C$ 可以表示为：
        当 $|\omega t - 2k\pi| < \theta_c$ 时，$i_C = i_{Cmax}\frac{\cos\omega t - \cos\theta_c}{1 - \cos\theta_c}$ 
        集电极电流可以分解为傅里叶级数： $i_C = \sum_{k=0}^{\infty}I_k\cos(k\omega t)$ 
        各次谐波的幅值 $I_k = i_{Cmax}\alpha_k(\theta_c)$，其中 $\alpha_k(\theta_c)$ 是波形分解系数，可以通过公式计算或查表得到。 

## 4.3 变频概述

**变频**：利用非线性器件的特性，将两个不同频率的信号进行频率变换，取出其中的和频或差频信号的过程。 
在超外差接收机中，变频是将接收到的高频信号 $v_s(f_s)$ 与本地振荡器产生的信号 $v_o(f_o)$ 在混频器中混合，得到一个固定的中频信号 $v_i(f_i)$。

![[屏幕截图 2025-05-06 224931 1.png|400]]
### 1. 混频器的功能 
混频器是实现频谱线性搬移的电路，可以看作一个三端口（六端）网络。 
混频器是频谱的线性搬移电路，完成频谱线性搬移功能的关键是获得两个输入信号的乘积项，具有这个乘积项，就可以实现所需的频谱线性搬移功能。
[Image: 调幅波示意图 11]
![[屏幕截图 2025-05-06 225413.png|400]]
* **输入信号**： 
    * 高频已调信号 $v_s(f_s)$ (例如来自天线的信号)
    * 本地振荡信号 $v_o(f_o)$ (由本地振荡器产生)
* **输出信号**： 
    * 中频信号 $v_I(f_I)$ (通常是一个固定的频率)

输入信号 $v_s$ 与输出信号 $v_I$ 的包络形状相同，频谱结构也相同，只是载波频率（填充频率）发生了改变。 
其中心频率关系为： $f_I = |f_o \pm f_s|$ 
* $f_I = f_o - f_s$：输出低中频，称为下变频 (这是超外差接收机中最常用的方式)。 
* $f_I = f_o + f_s$：输出高中频，称为上变频。 

### 2. 混频过程 
混频器实现频谱线性搬移的关键是获得两个输入信号的**乘积项**。 
理想的混频过程可以看作一个乘法器后接一个带通滤波器。 
[Image: 混频器一般结构框图和频谱示意图 12]
![[屏幕截图 2025-05-06 225615.png|400]]
输入信号 $V_s$ (频率 $\omega_s$) 和本地振荡信号 $V_o$ (频率 $\omega_o$) 通过非线性元件（或乘法器）后，产生包含 $\omega_o + \omega_s$ 和 $\omega_o - \omega_s$ 的分量。输出端的带通滤波器选择所需要的中频分量，例如 $\omega_I = \omega_o - \omega_s$。 
输出中频信号的包络形状没有变化，只是填充频率由 $\omega_s$ 变化为 $\omega_I$。 

### 3. 混频器的电路组成 
一个典型的混频电路通常包含：
* **输入回路**：选择所需的高频信号 $f_s$，并将其耦合到非线性器件。通常是调谐于 $f_s$ 的LC谐振回路。 
* **本地振荡器 (本振)**：产生稳定且频率准确的本地振荡信号 $f_o$ 。
* **非线性器件**：核心部分，利用其非线性特性实现频率变换，产生和频与差频分量。可以是二极管、三极管、场效应管等。 
* **输出选频网络**：从非线性器件的输出中选择所需的中频信号 $f_I$，抑制其他不需要的频率分量。通常是调谐于 $f_I$ 的LC谐振回路或晶体/陶瓷滤波器。 

### 4. 对变频器的要求 
理想的变频器应具备以下特性：
* **混频失真要小**：输出中频信号的包络应尽可能不失真地复制输入高频信号的包络。 
* **混频增益尤其是混频功率增益要高**：能够对信号进行有效放大，以提高接收机的灵敏度。 
* **混频器本身的噪声要尽可能小**：混频器通常位于接收机的前端，其噪声性能直接影响整个接收机的噪声系数。 
* **抗干扰性好**：能有效地抑制各种可能的混频干扰，如镜像频率干扰、中频干扰等。 

### 5. 混频器的主要性能指标 
* **变频增益**： 
    * 变频电压增益： $A_{vc} = \frac{V_I}{V_s}$ (输出中频电压有效值 $V_I$ 与输入高频信号电压有效值 $V_s$ 之比) 
    * 变频功率增益： $G_{Pc} = \frac{P_I}{P_s}$ (输出中频功率 $P_I$ 与输入高频信号功率 $P_s$ 之比) 
* **噪声系数 ($F_n$)**： 
    $F_n = \frac{(P_{si}/P_{ni})}{(P_{so}/P_{no})}$ (输入端信号与噪声功率比 $(P_{si}/P_{ni})$ / 输出端信号与噪声功率比 $(P_{so}/P_{no})$)。表示混频器使信噪比恶化的程度，希望 $F_n$ 越小越好。 
* **失真与干扰**： 
    * **频率失真**：指不同频率分量的增益或相移不同引起的失真。
    * **非线性失真**：由元件的非线性特性引起的，产生不需要的谐波和组合频率分量。 
* **选择性**：指中频输出回路对所需中频信号的选择能力，以及对邻近干扰信号的抑制能力。通常用矩形系数来衡量，希望矩形系数趋近于1。 

### 6. 变频器的分类 
* **按本振信号 $v_o(t)$ 来源分**： 
    * **自激式变频器**：本地振荡器与混频功能由同一个有源器件（如一个晶体管）完成，电路相对简单，但可能存在频率牵引问题。
    * **他激式变频器**：本地振荡器与混频器是相互独立的单元，稳定性较好。 
* **按器件分**： 
    * 二极管混频器
    * 三极管混频器
    * 场效应管混频器
    * 差分对管混频器
    * 集成模拟乘法器混频器
* **按电路结构分**： 
    * 单管混频器
    * 平衡混频器（如单平衡、双平衡/环形混频器），可以抑制某些不需要的频率分量，改善抗干扰性能。
    * 环形混频器 (一种双平衡混频器) 

## 4.4 晶体管混频器 

晶体管由于其非线性特性（主要是输入特性 $i_B=f(v_{BE})$ 或转移特性 $i_C=f(v_{BE})$ 的非线性）以及放大作用，常被用作混频器件。

**特点**： 
* 共射电路：$v_s$ 和 $v_o$ 都加在基极和发射极之间（直接或间接），利用 $i_C$ 与 $v_{BE}$ 的非线性关系进行频率变换。适用于频率较低（例如 $f < 100 \text{MHz}$）的场合，如广播、电视接收机。 
    * (a) 基极注入本振：优点是所需注入的本振功率较小，输入阻抗高，变频增益大，本振的负载较轻；但 $v_s$ 对 $v_o$ 有影响，容易产生频率牵引现象。 
    * (b) 射极注入本振：优点是 $v_s$ 和 $v_o$ 之间的互相影响小，不容易产生频率牵引现象；但要求本振输出较大功率。 
* 共基电路：高频特性优于共射电路，常用于较高频率（如 $100\text{MHz} \sim 300\text{MHz}$）的调频接收机。电路 (c) 和 (a) 类似，也容易产生频率牵引。 

**频率牵引**：指本振信号的频率 $f_o$ 受到输入信号 $f_s$ 或负载变化影响而发生偏移。

### 2. 工作原理和分析 (以图4.6.1(a)为例，基极输入信号，基极注入本振) 
[Image: 晶体管混频器工作原理分析电路图 18]
![[图片3.png|200]]
输入到基极的总电压为 $v_{BE} = V_{BQ} + v_s + v_o$，其中 $V_{BQ}$ 是直流偏置电压，$v_s = V_{sm}\cos\omega_s t$ 是高频信号，$v_o = V_{om}\cos\omega_o t$ 是本振信号。 

假设 $V_{om} \gg V_{sm}$，即本振信号远大于输入高频信号。 
集电极电流 $i_C$ 是 $v_{BE}$ 的非线性函数： $i_C = f(v_{BE})$。 
可以将 $v_{BE}$ 看作是在一个由直流偏置 $V_{BB}$ 和本振信号 $v_o$ 共同决定的动态工作点 $v_{BQ}(t) = V_{BB} + V_{om}\cos\omega_o t$ 上叠加了一个小信号 $v_s$。 
即 $v_{BE} = (V_{BB} + v_o) + v_s$。 
将 $i_C = f(v_{BE})$ 在 $v_{BE_0}(t) = V_{BB} + v_o$ 处进行泰勒级数展开： 
$i_C = f(V_{BB}+v_o) + f'(V_{BB}+v_o)v_s + \frac{1}{2}f''(V_{BB}+v_o)v_s^2 + \dots$ 
由于 $v_s$ 是小信号，可以忽略二次及以上各项，得到： 
$i_C \approx f(V_{BB}+v_o) + f'(V_{BB}+v_o)v_s$ 
[Image: 幂级数分析法图示 21, 22]
![[图片5.png|200]]

其中：
* $f(V_{BB}+v_o)$：是当 $v_{BE} = V_{BB}+v_o$ (即只有本振信号作用) 时的集电极电流，它是一个包含直流分量 $I_{c0}$、基波 $\omega_o$、各次谐波 $m\omega_o$ 的周期性电流：
    $f(V_{BB}+v_o) = I_{c0} + I_{cm1}\cos\omega_o t + I_{cm2}\cos 2\omega_o t + \dots$ 
* $f'(V_{BB}+v_o)$：是当 $v_{BE} = V_{BB}+v_o$ 时晶体管的**时变跨导**，用 $g_m(t)$ 或 $g(t)$ 表示。它也是一个随本振信号 $\omega_o$ 周期性变化的量：
    $g_m(t) = f'(V_{BB}+v_o) = g_0 + g_1\cos\omega_o t + g_2\cos 2\omega_o t + \dots$ 
    这里 $g_0, g_1, g_2, \dots$ 是 $g_m(t)$ 傅里叶级数展开的系数。

代入 $v_s = V_{sm}\cos\omega_s t$，则 $i_C$ 中包含 $v_s$ 的项为：
$g_m(t)v_s = (g_0 + g_1\cos\omega_o t + g_2\cos 2\omega_o t + \dots)(V_{sm}\cos\omega_s t)$ 
展开后得到：
$V_{sm}[g_0\cos\omega_s t + \frac{g_1}{2}\cos(\omega_o-\omega_s)t + \frac{g_1}{2}\cos(\omega_o+\omega_s)t + \frac{g_2}{2}\cos(2\omega_o-\omega_s)t + \frac{g_2}{2}\cos(2\omega_o+\omega_s)t + \dots]$ 
输出回路通过选频网络选择所需的差频（中频）分量，例如 $\omega_I = \omega_o - \omega_s$ (假设 $\omega_o > \omega_s$)。
则中频电流的幅值为 $I_{Im} = V_{sm}\frac{g_1}{2}$。 
其中 $g_1$ 是时变跨导 $g_m(t)$ 的基波分量幅值，称为**变频跨导**，通常用 $g_c$ 表示，即 $g_c = \frac{g_1}{2}$。 
可见，中频电流的幅值 $I_{Im}$ 与输入信号幅值 $V_{sm}$ 成正比，且与变频跨导 $g_c$ 成正比。混频过程保留了信号原来的频谱内部结构信息，只是载频发生了改变。 

### 4.4.3 变频器的计算 
主要计算变频跨导、输入电导和变频增益。
[Image: 晶体管混合Pi等效电路和简化转移等效电路 24, 25]
![[图片6.png|300]]![[图片7.png|300]]
1.  **变频跨导 $g_c$** 
    晶体管在信号频率 $\omega_s$ 下的跨导为：
    $g = \frac{\dot{I}_c}{\dot{V}_s} = \frac{g_m}{1+j\omega_s r_{bb'}C_{b'e}}$  (近似，基于简化的等效电路) 
    其中 $g_m = \frac{I_E}{V_T} \approx \frac{I_E}{26mV}$ (在室温下，$V_T$ 为热电压)。
    由于 $I_E$ (发射极电流) 受本振信号控制而周期性变化，所以 $g_m$ 也是时变的。
    变频跨导 $g_c$ 是这个时变跨导 $g(t)$ 的傅里叶级数中与 $\cos\omega_o t$ 相乘后得到的与 $\cos(\omega_o \pm \omega_s)t$ 相关的系数的一半，即 $g_c = \frac{1}{2}g_1$。 
    $|g_c| = \frac{1}{2} \frac{I_{E1}/(26mV)}{\sqrt{1+(\frac{\omega_s}{\omega_T} \cdot \frac{I_{E1}}{26mV} r_{bb'})^2}}$ (其中 $I_{E1}$ 是发射极电流基波幅值，$\omega_T$ 是晶体管的特征角频率)。近似计算较为复杂，常通过图解法或经验选取。 

2.  **变频输入电导 $g_{ic}$** 
    [Image: 晶体管混频器输入等效电路 26]
    ![[图片9.png|300]]
    考虑到输出中频回路对输入信号频率 $\omega_s$ 可看成短路，输入导纳 $y_{ie}$ 为：
    $y_{ie} = \frac{\dot{I}_b}{\dot{V}_s} = \frac{g_{b'e} + j\omega_s(C_{b'e}+C_{b'c})}{1+r_{bb'}[g_{b'e} + j\omega_s(C_{b'e}+C_{b'c})]}$ 
    近似得到输入电导 $g_{ic} \approx \frac{g_{b'e} + \omega_s^2 C_{b'e}^2 r_{bb'}}{1+(\omega_s C_{b'e} r_{bb'})^2}$ (忽略了 $C_{b'c}$ 且 $r_{bb'}g_{b'e} \ll 1$) 
    输入电导 $g_{ic}$ 也是随本振信号周期性变化的，通常取其平均值。

3.  **变频增益** 
    [Image: 晶体管混频器等效输出电路 27]
    
    ![[图片8.png|300]]
    * **电压增益 $A_{vc}$**：
        中频输出电压幅值 $V_{Im} = \frac{I_{Im}}{g_{oc}+G_L} = \frac{g_c V_{sm}}{g_{oc}+G_L}$ 
        $A_{vc} = \frac{V_{Im}}{V_{sm}} = \frac{g_c}{g_{oc}+G_L}$ 
        其中 $g_{oc}$ 是混频器输出电导， $G_L$ 是负载电导。
    * **功率增益 $G_{pc}$**：
        $G_{pc} = \frac{P_I}{P_s} = \frac{V_{Im}^2 G_L}{V_{sm}^2 g_{ic}} = (A_{vc})^2 \frac{G_L}{g_{ic}}$ 
        当输出匹配，即 $g_{oc}=G_L$ 时，可获得最大功率增益：
        $G_{pcmax} = \frac{g_c^2}{4g_{ic}g_{oc}}$ 

### 4.4.4 混频器工作状态的选择 
主要从变频增益、噪声和非线性失真（包括变频干扰）来考虑。
* **振荡电压 $V_{om}$ 的选择**： 
    * 对于硅管： $V_{om} = (150 \pm 50)\text{mV}$ 
    * 对于锗管： $V_{om} = (80 \pm 30)\text{mV}$ 
    * 一般常用晶体管的最佳振荡电压： $V_o = 50\text{mV} \sim 200\text{mV}$。本振电压 $V_{om}$ 过小，非线性作用弱，变频增益低；过大则可能导致晶体管进入饱和区或截止区过深，噪声增大，失真增加。 
* **发射极电流 $I_E$ 的选择**： 
    * 自激式： $(0.4 \sim 0.6)\text{mA}$ (文档中为0.4-0.0mA，应为笔误) 
    * 他激式： $(0.3 \sim 0.5)\text{mA}$ 
    * 通常： $I_E = (0.4 \sim 0.6)\text{mA}$。静态工作电流 $I_E$ 的选择需要平衡增益和噪声。 

**结论**：混频器的噪声系数 $F_n$ 一方面取决于混频器本身的噪声，另一方面还取决于变频功率增益 $G_{pc}$。混频器本身噪声愈小，变频功率增益 $G_{pc}$ 愈大，则混频器的噪声系数 $F_n$ 愈小。 

### 晶体管混频器的电路举例 
[Image: 晶体管自激式混频器电路图 30, 32]
![[图片10.png|400]]
这是一个典型的收音机自激式变频电路，将本振器与混频器功能合并在一个晶体管中。 
* **自激式还是他激式?** 自激式，因为本地振荡信号由同一晶体管及其周边元件（如Tr2变压器反馈）产生。

* **接收机电路还是发射机电路?** 接收机电路（“至中放”表明输出到中频放大器）。

## 4.5 二极管混频器和环形混频器 
在高质量通信设备中以及工作频率较高时，常使用二极管混频器，特别是平衡型混频器。
**优点**：噪声低，电路简单，组合频率分量相对较少（尤其平衡结构）。 

### 1. 单二极管混频电路 
[Image: 单二极管混频电路图 33]
![[图片11.png|200]]
利用二极管的非线性伏安特性（或其开关特性）进行混频。
**混频原理（开关函数分析法）**： 
**条件**： 
* 输入信号 $v_s(t) = V_{sm}\cos\omega_s t$ 为小信号。
* 本地振荡信号 $v_o(t) = V_{om}\cos\omega_o t$ 为振幅足够大的信号，即 $V_{om} \gg V_{sm}$。 
此时，二极管D的导通与截止主要受大信号 $v_o(t)$ 的控制，工作于开关状态。 
[Image: 二极管伏安特性曲线 34]
![[图片12.png|200]]
定义开关函数 $S(t)$： 
$S(t) = \begin{cases} 1 & (v_o(t) > V_{th}, \text{二极管导通}) \\ 0 & (v_o(t) < V_{th}, \text{二极管截止}) \end{cases}$
(理想情况下，导通阈值 $V_{th}=0$)
当 $v_o(t)$ 为正弦大信号时，$S(t)$ 是一个周期性方波（或接近方波），可以展开为傅里叶级数：
$S(t) = \frac{1}{2} + \frac{2}{\pi}\cos\omega_o t - \frac{2}{3\pi}\cos 3\omega_o t + \frac{2}{5\pi}\cos 5\omega_o t + \dots$ 
流过二极管的电流 $i(t)$ 可以近似表示为 $i(t) \approx g_d S(t) (v_s(t) + v_o(t) - v_i(t))$，其中 $g_d = 1/r_d$ 是二极管导通时的平均电导，$r_d$ 是导通电阻，$v_i(t)$ 是中频输出电压。 
（更简化地，若只考虑小信号 $v_s(t)$ 对电流的贡献，则 $i(t) \approx g_d S(t) v_s(t)$）。
将 $S(t)$ 和 $v_s(t)$ 代入，可得：
$i(t) = g_d \left(\frac{1}{2} + \frac{2}{\pi}\cos\omega_o t - \dots \right) (V_{sm}\cos\omega_s t)$
$i(t) = g_d \left[ \frac{1}{2}V_{sm}\cos\omega_s t + \frac{1}{\pi}V_{sm}\cos(\omega_o-\omega_s)t + \frac{1}{\pi}V_{sm}\cos(\omega_o+\omega_s)t + \dots \right]$ 
通过输出选频网络提取所需的中频分量，如 $(\omega_o-\omega_s)t$ 或 $(\omega_o+\omega_s)t$。
**特点**： 
* 缺点：电压传输系数较低，例如在全匹配条件下， $A_{vc} \approx \frac{1}{\pi}$。这意味着二极管混频器通常有损耗而不是增益。 
* 优点：产生的组合频率分量相对较少（相比于强非线性元件）。 

### 2. 二极管平衡混频器 
[Image: 二极管平衡混频器电路图 39]
![[图片13.png|300]]
由两个特性相同的二极管、输入变压器Tr1和输出变压器Tr2（或中频选频网络）组成。
本地振荡信号 $v_o$ 加在两个二极管的公共端和变压器的中心抽头之间。输入信号 $v_s$ 通过中心抽头的变压器Tr1对称地加到两个二极管上。
假设 $v_s$ 加到D1上为 $+\frac{v_s}{2}$，加到D2上为 $-\frac{v_s}{2}$ (相对于某个参考点)。
流过D1的电流 $i_1 = g_d S_1(t)(v_o + \frac{v_s-v_i}{2})$ 
流过D2的电流 $i_2 = g_d S_2(t)(v_o - \frac{v_s-v_i}{2})$ (注意 $S_1(t)$ 和 $S_2(t)$ 的相位关系取决于 $v_o$ 如何驱动D1和D2)
如果 $v_o$ 使D1和D2同相导通和截止 (即 $S_1(t)=S_2(t)=S(t)$)，且输出电流 $i = i_1-i_2$。 
则 $i = i_1 - i_2 = g_d S(t)(v_s - v_i)$ 
$i = g_d \left(\frac{1}{2} + \frac{2}{\pi}\cos\omega_o t - \dots \right) (V_{sm}\cos\omega_s t - V_{im}\cos\omega_i t)$ 
展开后，输出电流 $i$ 中将不包含本振信号 $v_o$ 及其谐波的直接分量，同时某些组合频率也会被抑制。
**特点**： 
1.  输出电流中不包含本振信号 $v_o$ ($\omega_o, 2\omega_o, \dots$) 的分量，以及与 $v_o$ 相关的偶次谐波与 $v_s$ 的组合频率。这有助于减少输出的无用频率成分，改善选择性。 
2.  等效电路和主要指标参数（如变频损耗）与单管时类似，但抗干扰能力增强。 

对比晶体管混频器和二极管平衡混频器的输出电流成分： 
* **二极管平衡混频器 (理想情况下)**：$i$ 中主要包含 $\omega_s, \omega_o \pm \omega_s, \dots$ (不含 $\omega_o, 2\omega_o, \dots$) 
* **晶体管混频器 (单管)**：$i_c$ 中包含 $\omega_o, 2\omega_o, \dots, \omega_s, \omega_o \pm \omega_s, 2\omega_o \pm \omega_s, \dots$ 

### 4.5.3 环形混频器 (双平衡混频器) 
也称为双平衡混频器，由四个特性相同的二极管接成桥式（环形）结构，并配合两个中心抽头的变压器。
**特点**：
* 能更好地抑制本振信号和输入信号 $v_s$ 在输出端的泄漏。
* 能抑制更多的无用组合频率分量，具有更好的频谱纯净度和更高的动态范围。
* 广泛应用于高性能接收机和测试设备中。

## 4.6 变频干扰 
由于混频器依赖非线性元件实现变频，其输出必然含有许多组合频率成分，形式为 $|\pm p f_o \pm q f_s|$，其中 $p, q = 0, 1, 2, \dots$。 
![[Screenshot_20250515_083335.jpg|450]]
[Image: 混频器干扰产生示意图 45, 46]
如果这些组合频率中的某些分量恰好等于或接近所需的中频频率 $f_I$ (通常 $f_I = |f_o - f_s|$，且带宽为 $B$)，它们就会和有用信号一起被中频滤波器选出，并被后续电路放大，从而引起干扰。 

主要干扰类型：
* **(1) 组合频率干扰 (自身组合干扰或干扰哨声)**：接收的有用射频信号 $v_s(f_s)$ 与本振信号 $v_o(f_o)$ 自身产生的组合频率落入中频通带内。 
    即 $|\pm p f_o \pm q f_s| = f_I \pm \frac{1}{2}B$。 
    当满足特定条件时，例如 $f_s = \frac{p \pm 1}{q - p} f_I$  ，就会产生干扰。
    **表现形式**：哨声。 
    **注意点**： 
    * 与外来干扰信号无关，不能仅靠提高前级电路（高放）的选择性来完全抑制。
    * **减少方法**： 
        * 正确选择中频频率 $f_I$，尽量避免低阶次（$p,q$ 较小）的强干扰。
        * 正确选择混频器的工作点，以减少不需要的组合频率分量的幅度。
        * 采用平衡混频器、环形混频器或乘法器等电路形式，从电路上抵消或抑制一些组合频率。
        * 适当减小信号幅度，或加入自动增益控制 (AGC) 电路。

* **(2) 副波道干扰 (或寄生波道干扰)**：外来的干扰信号 $v_n(f_n)$ 与本振信号 $v_o(f_o)$ 混频后，其组合频率落入中频通带内。 
    即 $|\pm p f_o \pm q f_n| = f_I \pm \frac{1}{2}B$。 
    可推导出干扰频率 $f_n = \frac{1}{q}|p f_o \pm f_I|$。 
    这类干扰中主要有中频干扰、镜频干扰等。 
    **表现形式**：串台（听到不希望的电台声音）、哨叫声。 

    * **中频干扰**： 
        当干扰信号频率 $f_n$ 恰好等于中频 $f_I$ 时 (对应 $p=0, q=1$)，即 $f_n = f_I$。 
        这种干扰信号如果能通过高放的选择性不足而进入混频器，就会直接通过混频器（即使混频器本身工作正常）进入中放电路。
        **抑制方法**： 
        * 提高混频器前级（高频放大器）的选择性，充分衰减频率为 $f_I$ 的干扰。
        * 在混频器前级增加中频陷波器（吸收电路）。
        * 合理选择中频数值，使中频频率选在接收机工作波段之外。

    * **镜像频率干扰 (镜频干扰)**： 
        这是最主要的一种副波道干扰。当 $p=1, q=1$ 时，若下变频 $f_I = f_o - f_s$ (且 $f_o > f_s$)，则干扰频率 $f_n$ 满足 $f_o - f_n = f_I$ 或 $f_n - f_o = f_I$。
        最常见的情况是 $f_n - f_o = f_I$，则 $f_n = f_o + f_I$。 
        将 $f_o = f_s + f_I$ 代入，得到 $f_n = (f_s + f_I) + f_I = f_s + 2f_I$。
        这个干扰频率 $f_n$ 与有用信号频率 $f_s$ 相对于本振频率 $f_o$ 是对称的，如同镜子内外一样。 
        ![[Screenshot_20250515_083548.jpg|300]]
        [Image: 镜像频率示意图，fs fo fn 在频率轴上的关系 50]
        **抑制方法**： 
        * **主要依靠提高混频器前级（高放）的选择性**，使得对镜像频率 $f_n$ 有足够的衰减。
        * **提高中频频率 $f_I$**：这样可以使镜像频率 $f_n = f_s + 2f_I$ 远离有用信号频率 $f_s$，从而更容易被高放的选择性电路滤除。例如，在需要宽调谐范围的接收机中，采用较高的第一中频。

* **(3) 交叉调制干扰 (交调干扰)**： 
    当一个强的干扰信号 $v_n(f_n)$ 与有用信号 $v_s(f_s)$ 同时作用于混频器的非线性输入端时，如果混频器的非线性特性包含三次或更高次项，强干扰信号的调制信息可能会转移（调制）到有用信号的载波上。 
    **现象**：当接收机调谐在有用信号 $f_s$ 时，能听到有用信号的声音，同时也能听到来自干扰信号 $f_n$ 的调制声音。若有用信号 $v_s$ 消失，则交调干扰也随之消失。 
    **特点**： 
    * 与有用信号并存，通过有用信号而起作用。
    * 与干扰信号的载频 $f_n$ 本身是否落入中频无关，任何频率的强干扰都可能形成交调。
    * 主要由混频器非线性特性的三次及更高次项产生。 
    **抑制措施**： 
    * 提高前级电路的选择性，以衰减强的带外干扰信号。
    * 选择线性度较好的混频器件（如动态范围大的混频器），或选择合适的工作点，使产生交调的三次及以上非线性项尽可能小。

* **(4) 互调干扰**： 
    当两个或多个强的干扰信号 $v_1(f_1)$ 和 $v_2(f_2)$ 同时作用于混频器的输入端时，它们之间会通过混频器的非线性作用产生新的组合频率，如 $|\pm m f_1 \pm n f_2|$。如果这些组合频率中的某个频率恰好等于或接近有用信号的频率 $f_s$ (或中频 $f_I$)，就会形成互调干扰。 
    例如，三阶互调产物 $2f_1 - f_2 = f_s$ 或 $2f_2 - f_1 = f_s$ 是常见的互调干扰。
    **表现形式**：啸叫声或杂乱的干扰声，可能没有有用信号的声音。 
    **减少方法**： 
    * 提高前级电路的选择性，以衰减强的带外干扰信号。
    * 选择具有良好线性度（高三阶截断点IP3）的混频器。使用平方律器件（其非线性主要为二次项）可以有效抑制奇数阶（如三阶）互调干扰。

* **(5) 阻塞干扰 (Blocking)**：一个非常强的临近频道或带外干扰信号，即使其频率本身不会产生直接的混频干扰，但由于其幅度过大，可能导致混频器（或前级放大器）进入饱和或截止状态，从而使其对有用信号的增益急剧下降，甚至无法正常工作。 

* **(6) 倒易混频 (Reciprocal Mixing)**：这是由本地振荡器信号的相位噪声（边带噪声）引起的。即使输入信号和干扰信号都处于带外，但如果本振信号存在较强的边带噪声，这些噪声会与强的带外干扰信号混频，产生落入中频的噪声，从而恶化有用信号的信噪比。 

### 互调干扰与交调干扰的区别 
* **作用对象**： 
    * 交调：干扰信号与有用信号之间的相互作用。
    * 互调：两个或多个干扰信号之间的相互作用。
* **听感**： 
    * 交调：检波后可以同时听到质量较差的有用信号和干扰信号的声音。
    * 互调：检波后主要听到啸叫声或随机的杂声干扰，有用信号可能被完全淹没。

### 副波道干扰和交叉调制干扰表现形式的区别 
* **副波道干扰**（如镜频干扰、中频干扰）：通常表现为串台（在一个电台的频率上收听到另一个电台的节目），或固定的哨叫声。 
* **交叉调制干扰**：表现为同时收听到两个电台的声音（有用信号和干扰信号的调制内容），且它们的强度变化通常是同步的（例如，有用信号变强时，交调进来的干扰声也可能变强）。 

### 关于二次变频 
手机等现代通信接收机中常采用二次变频（甚至多次变频）的结构：
* **第一次变频采用高中频 ($f_{I1}$)**： 
    * **优点**：可以有效地抑制镜像频率干扰。因为 $f_{mir1} = f_s + 2f_{I1}$，较高的 $f_{I1}$ 使得 $f_{mir1}$ 与 $f_s$ 相距很远，更容易被射频前端的滤波器（如天线双工器、射频带通滤波器）滤除。
    * **缺点**：高中频时，获得高的增益和窄的带宽（高选择性）相对困难且成本较高。
* **第二次变频采用低中频 ($f_{I2}$)**： 
    * **优点**：在较低的中频下，更容易实现高增益、高选择性（窄带滤波）和稳定的放大。现代数字信号处理也更倾向于在较低频率下进行A/D转换和处理。
    * **缺点**：如果直接使用低中频作为第一中频，则镜像频率 $f_{mir} = f_s + 2f_{I2}$ 会离 $f_s$ 很近，难以抑制。
通过二次变频，结合了高第一中频抑制镜像干扰和低第二中频易于实现高增益和选择性的优点。
